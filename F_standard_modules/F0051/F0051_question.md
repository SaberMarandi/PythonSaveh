# سوال F0051 - کار با JSON

## سطح: متوسط

## صورت مسئله
برنامه‌ای بنویسید که یک دیکشنری را به JSON تبدیل کند و چاپ کند.

## ورودی
نام و سن

## خروجی
JSON string

## مثال
```
ورودی:
Ali
25

خروجی:
{"name": "Ali", "age": 25}
```

## راهنمایی
- از ماژول json استفاده کنید
- از json.dumps() برای تبدیل استفاده کنید
