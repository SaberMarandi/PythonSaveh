# سوال F0005 - محاسبه جذر

## صورت مسئله
برنامه‌ای بنویسید که با استفاده از ماژول math، جذر یک عدد را محاسبه کند.

## ورودی
یک عدد اعشاری مثبت

## خروجی
جذر عدد (با 2 رقم اعشار)

## مثال
```
ورودی:
16

خروجی:
4.00
```

```
ورودی:
2

خروجی:
1.41
```

## راهنمایی
- از ماژول math استفاده کنید
- تابع math.sqrt() جذر را محاسبه می‌کند
- برای فرمت اعشار از f-string استفاده کنید
