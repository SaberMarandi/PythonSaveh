# سوال F0004 - استفاده از ماژول random

## صورت مسئله
برنامه‌ای بنویسید که با استفاده از ماژول random، n عدد تصادفی بین a و b تولید کند.

## ورودی
- خط اول: تعداد اعداد n
- خط دوم: حد پایین a
- خط سوم: حد بالا b

## خروجی
n عدد تصادفی (هر کدام در یک خط)

## مثال
```
ورودی:
5
1
10

خروجی:
3
7
2
9
5
```

## راهنمایی
- از random.randint(a, b) استفاده کنید
- از حلقه for برای تولید n عدد استفاده کنید
