# سوال F0004 - تولید عدد تصادفی

## صورت مسئله
برنامه‌ای بنویسید که با استفاده از ماژول random، یک عدد تصادفی بین a و b (شامل هر دو) تولید کند.

## ورودی
دو عدد صحیح a و b (a ≤ b)

## خروجی
یک عدد تصادفی بین a و b

## مثال
```
ورودی:
1
10

خروجی:
7
```

(خروجی هر بار متفاوت است)

## راهنمایی
- از ماژول random استفاده کنید
- تابع randint(a, b) عدد تصادفی بین a و b تولید می‌کند
- هر دو سر بازه شامل می‌شوند
