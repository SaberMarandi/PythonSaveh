# سوال C0071 - استفاده از map()

## صورت مسئله
برنامه‌ای بنویسید که از تابع map() برای محاسبه مربع اعداد یک لیست استفاده کند.

## ورودی
تعداد عناصر n، سپس n عدد صحیح

## خروجی
لیست مربع اعداد

## مثال
```
ورودی:
5
1 2 3 4 5

خروجی:
[1, 4, 9, 16, 25]
```

## راهنمایی
- map() یک تابع را روی تمام عناصر اعمال می‌کند
- نتیجه را به لیست تبدیل کنید
