# سوال C0028 - تابع فاکتوریل بازگشتی

## صورت مسئله
تابعی بازگشتی (Recursive) به نام factorial بنویسید که فاکتوریل عدد n را محاسبه کند.

## ورودی
یک عدد صحیح n (0 ≤ n ≤ 15)

## خروجی
فاکتوریل n

## مثال
```
ورودی:
5

خروجی:
120
```

## راهنمایی
- حالت پایه: factorial(0) = 1 و factorial(1) = 1
- حالت بازگشتی: factorial(n) = n * factorial(n-1)
- تابع خودش را صدا می‌زند
