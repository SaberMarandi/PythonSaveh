# سوال C0007 - تابع با پارامتر پیش‌فرض

## صورت مسئله
برنامه‌ای بنویسید که یک تابع برای محاسبه مساحت مستطیل بنویسد. اگر فقط یک پارامتر داده شود، مربع در نظر گرفته شود.

## ورودی
- خط اول: طول
- خط دوم: عرض (اختیاری، اگر نباشد با طول برابر است)

## خروجی
مساحت

## مثال
```
ورودی:
5
3

خروجی:
15
```

## راهنمایی
- از پارامتر پیش‌فرض استفاده کنید
- def area(length, width=None)
