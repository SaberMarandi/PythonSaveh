# سوال C0010 - تابع lambda

## صورت مسئله
برنامه‌ای بنویسید که با استفاده از تابع lambda، یک لیست از اعداد را مرتب کند بر اساس باقیمانده تقسیم بر 5.

## ورودی
- خط اول: تعداد اعداد n
- خط دوم: n عدد صحیح (با فاصله)

## خروجی
لیست مرتب شده (با فاصله)

## مثال
```
ورودی:
5
12 7 23 18 5

خروجی:
5 23 12 7 18
```

## راهنمایی
- از sorted() با key=lambda استفاده کنید
- lambda x: x % 5
