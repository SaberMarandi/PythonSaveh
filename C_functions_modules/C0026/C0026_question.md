# سوال C0026 - تابع محاسبه توان

## صورت مسئله
تابعی به نام power بنویسید که توان n‌ام عدد x را محاسبه و برگرداند.

## ورودی
دو عدد صحیح x و n

## خروجی
x به توان n

## مثال
```
ورودی:
2
3

خروجی:
8
```

توضیح: 2³ = 8

## راهنمایی
- از عملگر ** برای محاسبه توان استفاده کنید
- یا از تابع pow()
- تابع باید دو پارامتر داشته باشد
