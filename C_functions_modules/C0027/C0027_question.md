# سوال C0027 - تابع بررسی زوج/فرد

## صورت مسئله
تابعی به نام is_even بنویسید که بررسی کند عدد ورودی زوج است یا فرد.

## ورودی
یک عدد صحیح

## خروجی
"Even" اگر زوج باشد، "Odd" اگر فرد باشد

## مثال
```
ورودی:
8

خروجی:
Even
```

```
ورودی:
7

خروجی:
Odd
```

## راهنمایی
- تابع باید True یا False برگرداند
- از عملگر % برای بررسی باقیمانده استفاده کنید
- عدد زوج بر 2 بخش‌پذیر است
