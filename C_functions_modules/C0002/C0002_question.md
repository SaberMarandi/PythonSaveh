# سوال C0002 - تابع با چند پارامتر

## صورت مسئله
برنامه‌ای بنویسید که یک تابع تعریف کند که سه عدد را دریافت کرده و بزرگترین آن‌ها را برگرداند.

## ورودی
سه عدد صحیح

## خروجی
بزرگترین عدد

## مثال
```
ورودی:
5
12
8

خروجی:
12
```

## راهنمایی
- از تابع `max()` استفاده کنید
- یا با if-else مقایسه کنید
