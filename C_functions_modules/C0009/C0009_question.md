# سوال C0009 - تابع فیبوناچی بازگشتی

## صورت مسئله
برنامه‌ای بنویسید که یک تابع بازگشتی برای محاسبه nامین عدد فیبوناچی تعریف کند.

## ورودی
یک عدد صحیح n

## خروجی
nامین عدد فیبوناچی

## مثال
```
ورودی:
7

خروجی:
13
```

## راهنمایی
- دنباله فیبوناچی: 0, 1, 1, 2, 3, 5, 8, 13, ...
- fib(n) = fib(n-1) + fib(n-2)
- شرط پایان: fib(0) = 0, fib(1) = 1
