# سوال C0006 - تابع بدون return

## صورت مسئله
برنامه‌ای بنویسید که یک تابع تعریف کند که یک پیام را چاپ می‌کند ولی چیزی برنمی‌گرداند.

## ورودی
یک رشته

## خروجی
پیام چاپ شده

## مثال
```
ورودی:
Hello

خروجی:
Message: Hello
```

## راهنمایی
- توابع بدون return مقدار None برمی‌گردانند
- از print() در داخل تابع استفاده کنید
