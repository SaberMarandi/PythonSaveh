# سوال C0003 - تابع بازگشتی

## صورت مسئله
برنامه‌ای بنویسید که یک تابع بازگشتی برای محاسبه فاکتوریل تعریف کند.

## ورودی
یک عدد صحیح غیرمنفی n

## خروجی
فاکتوریل n

## مثال
```
ورودی:
5

خروجی:
120
```

## راهنمایی
- تابع بازگشتی خودش را صدا می‌زند
- شرط پایان: n == 0 یا n == 1
- فاکتوریل n = n × فاکتوریل(n-1)
