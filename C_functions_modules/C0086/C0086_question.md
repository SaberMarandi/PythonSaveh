# سوال C0086 - استفاده از map

## صورت مسئله
برنامه‌ای بنویسید که با استفاده از تابع map، تمام اعداد یک لیست را به توان 2 برساند.

## ورودی
- خط اول: تعداد اعداد n
- خط دوم: n عدد صحیح (با فاصله)

## خروجی
لیست اعداد به توان 2 (با فاصله)

## مثال
```
ورودی:
5
2 3 4 5 6

خروجی:
4 9 16 25 36
```

## راهنمایی
- از تابع map() استفاده کنید
- map(lambda x: x**2, numbers)
- نتیجه را به لیست تبدیل کنید
