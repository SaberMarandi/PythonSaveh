# سوال E0062 - پلی‌مورفیسم با حیوانات

## سطح: متوسط

## صورت مسئله
برنامه‌ای بنویسید که لیستی از حیوانات مختلف ایجاد کند و متد make_sound را فراخوانی کند.

## ورودی
- خط اول: تعداد حیوانات
- برای هر حیوان: نوع و نام

## خروجی
صدای هر حیوان

## مثال
```
ورودی:
2
dog
Max
cat
Lucy

خروجی:
Max: Woof!
Lucy: Meow!
```

## راهنمایی
- از لیست برای نگهداری اشیاء استفاده کنید
