# سوال E0054 - وراثت با اشکال هندسی

## سطح: متوسط

## صورت مسئله
برنامه‌ای بنویسید که کلاس‌های Rectangle و Circle از کلاس Shape ارث‌بری کنند.

## ورودی
- خط اول: نوع شکل (rectangle یا circle)
- خط دوم: رنگ
- خطوط بعدی: ابعاد (برای rectangle: طول و عرض، برای circle: شعاع)

## خروجی
- رنگ
- مساحت (با دو رقم اعشار)

## مثال
```
ورودی:
rectangle
Red
5
3

خروجی:
رنگ: Red
مساحت: 15.00
```

## راهنمایی
- کلاس پایه Shape دارای رنگ است
- هر کلاس فرزند متد area را پیاده‌سازی می‌کند
