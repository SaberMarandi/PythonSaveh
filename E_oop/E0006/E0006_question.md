# سوال E0006 - متد با پارامتر

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Calculator با متد add برای جمع دو عدد ایجاد کند.

## ورودی
- خط اول: عدد اول
- خط دوم: عدد دوم

## خروجی
حاصل جمع دو عدد

## مثال
```
ورودی:
10
20

خروجی:
30
```

## راهنمایی
- متد add دو پارامتر دریافت می‌کند
- از self برای تعریف متد استفاده کنید
