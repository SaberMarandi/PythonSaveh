# سوال E0018 - کلاس لیست دانشجویان

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس StudentList برای مدیریت لیست دانشجویان ایجاد کند.

## ورودی
- خط اول: تعداد دانشجویان
- خطوط بعدی: نام دانشجویان

## خروجی
تعداد دانشجویان و لیست آن‌ها

## مثال
```
ورودی:
3
Ali
Sara
Reza

خروجی:
تعداد: 3
Ali
Sara
Reza
```

## راهنمایی
- از لیست برای نگهداری نام‌ها استفاده کنید
- متد add_student برای افزودن دانشجو بسازید
