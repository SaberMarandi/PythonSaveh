# سوال E0050 - Property برای BMI

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس BMI با property برای محاسبه شاخص توده بدنی ایجاد کند.

## ورودی
- خط اول: وزن (کیلوگرم)
- خط دوم: قد (متر)

## خروجی
- وزن
- قد
- BMI (با دو رقم اعشار)
- دسته‌بندی

## مثال
```
ورودی:
70
1.75

خروجی:
وزن: 70.0 کیلوگرم
قد: 1.75 متر
BMI: 22.86
دسته‌بندی: نرمال
```

## راهنمایی
- BMI = وزن / (قد²)
- کم‌وزن: BMI < 18.5
- نرمال: 18.5 <= BMI < 25
- اضافه‌وزن: 25 <= BMI < 30
- چاق: BMI >= 30
