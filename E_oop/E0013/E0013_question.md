# سوال E0013 - کلاس با Validation

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Age ایجاد کند که سن را فقط در صورت معتبر بودن (0 تا 150) قبول کند.

## ورودی
سن

## خروجی
- اگر سن معتبر باشد: "سن معتبر: [سن]"
- اگر سن نامعتبر باشد: "سن نامعتبر"

## مثال 1
```
ورودی:
25

خروجی:
سن معتبر: 25
```

## مثال 2
```
ورودی:
200

خروجی:
سن نامعتبر
```

## راهنمایی
- در __init__ اعتبارسنجی انجام دهید
- از شرط برای بررسی محدوده استفاده کنید
