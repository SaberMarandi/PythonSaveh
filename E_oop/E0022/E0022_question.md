# سوال E0022 - شمارش اشیاء با متغیر کلاس

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Employee ایجاد کند و تعداد کارمندان را با متغیر کلاس شمارش کند.

## ورودی
- خط اول: تعداد کارمندان
- خطوط بعدی: نام کارمندان

## خروجی
- تعداد کل کارمندان
- نام هر کارمند

## مثال
```
ورودی:
3
Ali
Sara
Reza

خروجی:
تعداد کارمندان: 3
Ali
Sara
Reza
```

## راهنمایی
- متغیر کلاس count را در __init__ افزایش دهید
