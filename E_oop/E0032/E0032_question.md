# سوال E0032 - متد __repr__

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Book با متدهای __str__ و __repr__ ایجاد کند.

## ورودی
- خط اول: عنوان کتاب
- خط دوم: نویسنده

## خروجی
- خط اول: خروجی __str__
- خط دوم: خروجی __repr__

## مثال
```
ورودی:
Python Guide
John Doe

خروجی:
Python Guide by John Doe
Book('Python Guide', 'John Doe')
```

## راهنمایی
- __str__ برای نمایش کاربرپسند
- __repr__ برای نمایش توسعه‌دهنده
