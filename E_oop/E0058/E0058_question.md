# سوال E0058 - محصولات مختلف

## سطح: متوسط

## صورت مسئله
برنامه‌ای بنویسید که کلاس‌های Book و Electronics از Product ارث‌بری کنند.

## ورودی
- خط اول: نوع محصول (book یا electronics)
- خط دوم: نام
- خط سوم: قیمت
- خطوط بعدی: اطلاعات اختصاصی

## خروجی
اطلاعات کامل محصول

## مثال
```
ورودی:
book
Python Guide
500000
John Doe
350

خروجی:
Python Guide: 500000.0 تومان, نویسنده: John Doe, صفحات: 350
```

## راهنمایی
- Book دارای نویسنده و تعداد صفحات است
- Electronics دارای برند و گارانتی است
