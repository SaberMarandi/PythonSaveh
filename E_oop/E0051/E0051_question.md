# سوال E0051 - وراثت ساده

## سطح: متوسط

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Animal و یک کلاس Dog که از Animal ارث‌بری می‌کند، تعریف کند.

## ورودی
نام حیوان

## خروجی
صدای حیوان

## مثال
```
ورودی:
Buddy

خروجی:
Buddy says: Woof!
```

## راهنمایی
- از کلمه کلیدی class و پرانتز برای وراثت استفاده کنید
- متد speak را override کنید
