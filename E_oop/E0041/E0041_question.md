# سوال E0041 - Getter و Setter

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Person با متدهای getter و setter برای نام و سن ایجاد کند.

## ورودی
- خط اول: نام اولیه
- خط دوم: سن اولیه
- خط سوم: نام جدید
- خط چهارم: سن جدید

## خروجی
- خط اول: اطلاعات اولیه
- خط دوم: اطلاعات جدید

## مثال
```
ورودی:
Ali
25
Ahmad
30

خروجی:
نام: Ali, سن: 25
نام: Ahmad, سن: 30
```

## راهنمایی
- از متغیرهای خصوصی استفاده کنید
- متدهای get برای خواندن
- متدهای set برای نوشتن
