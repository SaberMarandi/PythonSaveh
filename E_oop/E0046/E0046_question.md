# سوال E0046 - Property با Validation

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Student با property برای نمره و وضعیت ایجاد کند.

## ورودی
- خط اول: نام
- خط دوم: نمره اولیه
- خط سوم: نمره جدید

## خروجی
- اطلاعات اولیه (نام، نمره، وضعیت)
- اطلاعات جدید (نمره، وضعیت)

## مثال
```
ورودی:
Ali
15
8

خروجی:
نام: Ali
نمره: 15.0
وضعیت: قبول
نمره: 8.0
وضعیت: مردود
```

## راهنمایی
- نمره باید بین 0 تا 20 باشد
- نمره >= 10: قبول
- نمره < 10: مردود
