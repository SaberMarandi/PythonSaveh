# سوال E0040 - کلاس کتابخانه

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Library برای مدیریت کتاب‌ها ایجاد کند.

## ورودی
- خط اول: نام کتابخانه
- خط دوم: تعداد کتاب‌ها
- خطوط بعدی: عنوان کتاب

## خروجی
- نام کتابخانه
- تعداد کتاب‌ها
- لیست کتاب‌ها (مرتب شده)

## مثال
```
ورودی:
City Library
3
Python Programming
Data Science
Web Development

خروجی:
کتابخانه: City Library
تعداد کتاب‌ها: 3
کتاب‌ها:
- Data Science
- Python Programming
- Web Development
```

## راهنمایی
- از لیست برای نگهداری کتاب‌ها استفاده کنید
- لیست را مرتب کنید
