# سوال E0039 - کلاس رستوران

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Restaurant برای مدیریت سفارشات ایجاد کند.

## ورودی
- خط اول: نام رستوران
- خط دوم: تعداد سفارشات
- خطوط بعدی: نام غذا و قیمت (با فاصله)

## خروجی
- نام رستوران
- تعداد سفارشات
- مجموع قیمت

## مثال
```
ورودی:
FastFood
3
Pizza 50000
Burger 35000
Fries 15000

خروجی:
رستوران: FastFood
تعداد سفارشات: 3
مجموع: 100000 تومان
```

## راهنمایی
- از لیست برای نگهداری سفارشات استفاده کنید
