# سوال E0021 - متغیر کلاس

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس School با متغیر کلاس برای نام مدرسه ایجاد کند.

## ورودی
- خط اول: نام مدرسه
- خط دوم: تعداد دانشجویان برای ایجاد
- خطوط بعدی: نام دانشجویان

## خروجی
- خط اول: نام مدرسه
- خطوط بعدی: نام دانشجویان

## مثال
```
ورودی:
Shahid Beheshti
2
Ali
Sara

خروجی:
مدرسه: Shahid Beheshti
دانشجو: Ali
دانشجو: Sara
```

## راهنمایی
- متغیر کلاس برای همه اشیاء مشترک است
- متغیر instance برای هر شیء جداگانه است
