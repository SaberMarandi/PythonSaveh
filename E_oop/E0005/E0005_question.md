# سوال E0005 - کلاس دایره

## صورت مسئله
کلاسی به نام Circle بنویسید که مساحت و محیط یک دایره را محاسبه کند.

کلاس باید شامل:
- Constructor که شعاع را دریافت کند
- متد area() برای محاسبه مساحت
- متد circumference() برای محاسبه محیط

## ورودی
یک عدد اعشاری: شعاع دایره

## خروجی
مساحت و محیط دایره (با 2 رقم اعشار)

## مثال
```
ورودی:
5

خروجی:
Area: 78.54, Circumference: 31.42
```

## راهنمایی
- مساحت = π × r²
- محیط = 2 × π × r
- از math.pi برای عدد پی استفاده کنید
