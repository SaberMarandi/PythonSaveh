# سوال E0005 - کلاس با چند Attribute

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Student با سه attribute (نام، سن، نمره) تعریف کند و اطلاعات دانشجو را نمایش دهد.

## ورودی
- خط اول: نام
- خط دوم: سن
- خط سوم: نمره

## خروجی
اطلاعات دانشجو به فرمت: "نام: [نام], سن: [سن], نمره: [نمره]"

## مثال
```
ورودی:
Ali
20
18.5

خروجی:
نام: Ali, سن: 20, نمره: 18.5
```

## راهنمایی
- از `__init__` برای مقداردهی اولیه استفاده کنید
- می‌توانید یک متد display برای نمایش اطلاعات بسازید
