# سوال E0030 - کلاس فروشگاه

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Shop با لیست محصولات و متدهای افزودن و نمایش ایجاد کند.

## ورودی
- خط اول: نام فروشگاه
- خط دوم: تعداد محصولات
- خطوط بعدی: نام محصولات

## خروجی
- نام فروشگاه
- تعداد محصولات
- لیست محصولات

## مثال
```
ورودی:
SuperMarket
3
Milk
Bread
Cheese

خروجی:
فروشگاه: SuperMarket
تعداد محصولات: 3
- Milk
- Bread
- Cheese
```

## راهنمایی
- از لیست برای نگهداری محصولات استفاده کنید
