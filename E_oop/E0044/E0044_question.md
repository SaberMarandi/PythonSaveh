# سوال E0044 - Property Decorator

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Rectangle با property decorator برای طول، عرض و مساحت ایجاد کند.

## ورودی
- خط اول: طول اولیه
- خط دوم: عرض اولیه
- خط سوم: طول جدید
- خط چهارم: عرض جدید

## خروجی
- خط اول: مساحت اولیه
- خط دوم: مساحت جدید

## مثال
```
ورودی:
5
3
10
4

خروجی:
مساحت: 15
مساحت: 40
```

## راهنمایی
- از @property برای getter استفاده کنید
- از @attribute.setter برای setter استفاده کنید
