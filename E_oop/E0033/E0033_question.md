# سوال E0033 - کلاس نقطه با __str__

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Point با متد __str__ برای نمایش مختصات ایجاد کند.

## ورودی
- خط اول: مختصات x
- خط دوم: مختصات y

## خروجی
نقطه به فرمت: "(x, y)"

## مثال
```
ورودی:
3
4

خروجی:
(3, 4)
```

## راهنمایی
- از __str__ برای فرمت‌بندی خروجی استفاده کنید
