# سوال E0014 - کلاس نقطه

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Point برای نقطه در صفحه مختصات ایجاد کند و فاصله آن از مبدأ را محاسبه کند.

## ورودی
- خط اول: مختصات x
- خط دوم: مختصات y

## خروجی
فاصله از مبدأ (با دو رقم اعشار)

## مثال
```
ورودی:
3
4

خروجی:
5.00
```

## راهنمایی
- فاصله از مبدأ: √(x² + y²)
- از math.sqrt استفاده کنید
