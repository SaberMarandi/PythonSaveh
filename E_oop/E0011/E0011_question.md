# سوال E0011 - Constructor با مقدار پیش‌فرض

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Car با constructor دارای مقدار پیش‌فرض برای رنگ ایجاد کند.

## ورودی
- خط اول: نام خودرو
- خط دوم: "yes" اگر رنگ وارد می‌شود، "no" در غیر این صورت
- خط سوم (اختیاری): رنگ

## خروجی
اطلاعات خودرو

## مثال 1
```
ورودی:
Toyota
yes
Red

خروجی:
نام: Toyota, رنگ: Red
```

## مثال 2
```
ورودی:
Honda
no

خروجی:
نام: Honda, رنگ: White
```

## راهنمایی
- مقدار پیش‌فرض رنگ "White" است
- از پارامتر پیش‌فرض در __init__ استفاده کنید
