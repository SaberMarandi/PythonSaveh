# سوال E0047 - Property برای قیمت نهایی

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Product با property برای قیمت نهایی با تخفیف ایجاد کند.

## ورودی
- خط اول: نام محصول
- خط دوم: قیمت
- خط سوم: درصد تخفیف

## خروجی
- نام محصول
- قیمت
- تخفیف
- قیمت نهایی (با دو رقم اعشار)

## مثال
```
ورودی:
Laptop
10000000
15

خروجی:
محصول: Laptop
قیمت: 10000000.0
تخفیف: 15.0%
قیمت نهایی: 8500000.00
```

## راهنمایی
- قیمت نهایی = قیمت × (1 - تخفیف/100)
