# سوال E0004 - کلاس مستطیل

## صورت مسئله
کلاسی به نام Rectangle بنویسید که مساحت و محیط یک مستطیل را محاسبه کند.

کلاس باید شامل:
- Constructor که طول و عرض را دریافت کند
- متد area() برای محاسبه مساحت
- متد perimeter() برای محاسبه محیط

## ورودی
دو عدد صحیح: عرض و ارتفاع

## خروجی
مساحت و محیط مستطیل

## مثال
```
ورودی:
5
3

خروجی:
Area: 15, Perimeter: 16
```

## راهنمایی
- مساحت = طول × عرض
- محیط = 2 × (طول + عرض)
- از __init__ برای مقداردهی اولیه استفاده کنید
