# سوال E0004 - متدهای کلاس

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Rectangle با متدهای area و perimeter ایجاد کند.

## ورودی
- خط اول: طول
- خط دوم: عرض

## خروجی
- خط اول: مساحت
- خط دوم: محیط

## مثال
```
ورودی:
5
3

خروجی:
15
16
```

## راهنمایی
- متد area: length * width
- متد perimeter: 2 * (length + width)
- از self برای دسترسی به ویژگی‌ها استفاده کنید
