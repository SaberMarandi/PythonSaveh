# سوال E0026 - کلاس ماشین حساب پیشرفته

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Calculator با متدهای مختلف ریاضی ایجاد کند.

## ورودی
- خط اول: عدد اول
- خط دوم: عدد دوم
- خط سوم: عملیات (add, sub, mul, div, pow, mod)

## خروجی
نتیجه عملیات

## مثال
```
ورودی:
10
3
pow

خروجی:
1000
```

## راهنمایی
- متد pow برای توان
- متد mod برای باقیمانده تقسیم
