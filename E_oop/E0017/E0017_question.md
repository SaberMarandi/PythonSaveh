# سوال E0017 - کلاس مستطیل با مساحت و محیط

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Rectangle ایجاد کند و بررسی کند آیا مربع است یا نه.

## ورودی
- خط اول: طول
- خط دوم: عرض

## خروجی
- اگر مربع باشد: "مربع"
- اگر مربع نباشد: "مستطیل"

## مثال 1
```
ورودی:
5
5

خروجی:
مربع
```

## مثال 2
```
ورودی:
5
3

خروجی:
مستطیل
```

## راهنمایی
- اگر طول و عرض برابر باشند، شکل مربع است
