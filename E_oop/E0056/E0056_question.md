# سوال E0056 - حساب پس‌انداز

## سطح: متوسط

## صورت مسئله
برنامه‌ای بنویسید که کلاس SavingsAccount از BankAccount ارث‌بری کند و سود اضافه کند.

## ورودی
- خط اول: شماره حساب
- خط دوم: موجودی اولیه
- خط سوم: نرخ سود (درصد)

## خروجی
- موجودی اولیه
- سود
- موجودی نهایی

## مثال
```
ورودی:
123456
10000000
5

خروجی:
موجودی اولیه: 10000000.00
سود: 500000.00
موجودی نهایی: 10500000.00
```

## راهنمایی
- سود = موجودی × نرخ سود / 100
