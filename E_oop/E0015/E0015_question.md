# سوال E0015 - کلاس دما

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Temperature برای تبدیل دما بین سلسیوس و فارنهایت ایجاد کند.

## ورودی
- خط اول: دما به سلسیوس

## خروجی
- خط اول: دما به فارنهایت (با دو رقم اعشار)

## مثال
```
ورودی:
0

خروجی:
32.00
```

## راهنمایی
- فرمول تبدیل: F = (C × 9/5) + 32
