# سوال E0035 - کلاس دانشجو با __str__ و __repr__

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Student با متدهای __str__ و __repr__ ایجاد کند.

## ورودی
- خط اول: نام
- خط دوم: شماره دانشجویی
- خط سوم: نمره

## خروجی
- خط اول: خروجی __str__
- خط دوم: خروجی __repr__

## مثال
```
ورودی:
Ali
12345
18.5

خروجی:
دانشجو: Ali (12345) - نمره: 18.5
Student('Ali', 12345, 18.5)
```

## راهنمایی
- __str__ برای نمایش فارسی و خوانا
- __repr__ برای نمایش تکنیکال
