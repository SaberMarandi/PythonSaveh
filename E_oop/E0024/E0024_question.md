# سوال E0024 - کلاس با متغیر خصوصی

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس BankAccount با متغیر خصوصی برای موجودی ایجاد کند.

## ورودی
- خط اول: موجودی اولیه
- خط دوم: مبلغ واریز
- خط سوم: مبلغ برداشت

## خروجی
موجودی نهایی

## مثال
```
ورودی:
1000
500
200

خروجی:
1300
```

## راهنمایی
- از __ برای متغیر خصوصی استفاده کنید
- متدهای عمومی برای دسترسی بسازید
