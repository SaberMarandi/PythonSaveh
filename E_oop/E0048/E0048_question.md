# سوال E0048 - Property برای حقوق

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Employee با property برای محاسبه پاداش و حقوق کل ایجاد کند.

## ورودی
- خط اول: نام کارمند
- خط دوم: حقوق پایه
- خط سوم: درصد پاداش

## خروجی
- نام کارمند
- حقوق پایه
- پاداش
- حقوق کل

## مثال
```
ورودی:
Ali
10000000
20

خروجی:
کارمند: Ali
حقوق پایه: 10000000
پاداش: 2000000
حقوق کل: 12000000
```

## راهنمایی
- پاداش = حقوق پایه × درصد پاداش / 100
- حقوق کل = حقوق پایه + پاداش
