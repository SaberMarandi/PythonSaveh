# سوال E0012 - شمارنده اشیاء

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Counter ایجاد کند که تعداد اشیاء ساخته شده را شمارش کند.

## ورودی
تعداد اشیاء برای ساخت

## خروجی
تعداد کل اشیاء ساخته شده

## مثال
```
ورودی:
5

خروجی:
5
```

## راهنمایی
- از متغیر کلاس برای شمارش استفاده کنید
- در __init__ شمارنده را افزایش دهید
