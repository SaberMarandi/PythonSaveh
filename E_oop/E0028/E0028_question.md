# سوال E0028 - کلاس مثلث

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Triangle برای محاسبه مساحت و محیط مثلث ایجاد کند.

## ورودی
- خط اول: ضلع اول
- خط دوم: ضلع دوم
- خط سوم: ضلع سوم

## خروجی
- خط اول: محیط
- خط دوم: مساحت (با دو رقم اعشار)

## مثال
```
ورودی:
3
4
5

خروجی:
12
6.00
```

## راهنمایی
- محیط = a + b + c
- مساحت = √(s(s-a)(s-b)(s-c)) که s = محیط/2
- از فرمول هرون استفاده کنید
