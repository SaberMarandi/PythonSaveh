# سوال E0031 - متد __str__

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Person با متد __str__ برای نمایش اطلاعات ایجاد کند.

## ورودی
- خط اول: نام
- خط دوم: سن

## خروجی
اطلاعات شخص به فرمت: "Person(name=[نام], age=[سن])"

## مثال
```
ورودی:
Ali
25

خروجی:
Person(name=Ali, age=25)
```

## راهنمایی
- متد __str__ برای نمایش خوانا استفاده می‌شود
- هنگام print شدن شیء، این متد فراخوانی می‌شود
