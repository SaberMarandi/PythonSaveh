# سوال E0025 - کلاس با ID خودکار

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس User ایجاد کند که به هر کاربر یک ID خودکار اختصاص دهد.

## ورودی
- خط اول: تعداد کاربران
- خطوط بعدی: نام کاربران

## خروجی
ID و نام هر کاربر

## مثال
```
ورودی:
3
Ali
Sara
Reza

خروجی:
ID: 1, نام: Ali
ID: 2, نام: Sara
ID: 3, نام: Reza
```

## راهنمایی
- از متغیر کلاس برای شمارش ID استفاده کنید
- در __init__ به هر شیء ID اختصاص دهید
