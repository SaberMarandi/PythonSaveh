# سوال E0020 - کلاس زمان

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Time برای نمایش زمان (ساعت، دقیقه، ثانیه) ایجاد کند.

## ورودی
- خط اول: ساعت
- خط دوم: دقیقه
- خط سوم: ثانیه

## خروجی
زمان به فرمت: HH:MM:SS

## مثال
```
ورودی:
14
30
45

خروجی:
14:30:45
```

## راهنمایی
- از f-string برای فرمت‌بندی استفاده کنید
- اعداد را با دو رقم نمایش دهید (02 به جای 2)
