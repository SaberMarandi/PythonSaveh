# سوال E0010 - کلاس کتاب

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Book با attributes (عنوان، نویسنده، سال انتشار) و متد display_info ایجاد کند.

## ورودی
- خط اول: عنوان کتاب
- خط دوم: نام نویسنده
- خط سوم: سال انتشار

## خروجی
اطلاعات کتاب به فرمت: "عنوان: [عنوان], نویسنده: [نویسنده], سال: [سال]"

## مثال
```
ورودی:
Python Programming
John Smith
2020

خروجی:
عنوان: Python Programming, نویسنده: John Smith, سال: 2020
```

## راهنمایی
- از __init__ برای مقداردهی اولیه استفاده کنید
- متد display_info اطلاعات را برمی‌گرداند
