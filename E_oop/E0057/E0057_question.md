# سوال E0057 - کارمندان با حقوق متفاوت

## سطح: متوسط

## صورت مسئله
برنامه‌ای بنویسید که کلاس‌های Manager و Developer از Employee ارث‌بری کنند.

## ورودی
- خط اول: نوع کارمند (manager یا developer)
- خط دوم: نام
- خط سوم: حقوق پایه
- خط چهارم: پاداش (برای manager) یا تعداد پروژه‌ها (برای developer)

## خروجی
- نام کارمند
- حقوق کل

## مثال
```
ورودی:
manager
Ali
10000000
2000000

خروجی:
کارمند: Ali
حقوق: 12000000
```

## راهنمایی
- Manager: حقوق کل = حقوق پایه + پاداش
- Developer: حقوق کل = حقوق پایه + (تعداد پروژه × 1000000)
