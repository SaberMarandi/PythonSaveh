# سوال E0043 - Getter و Setter برای تبدیل

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Temperature با getter و setter برای سلسیوس و فارنهایت ایجاد کند.

## ورودی
- خط اول: دما به سلسیوس
- خط دوم: دما به فارنهایت (برای تنظیم)

## خروجی
- دو خط اول: دما به سلسیوس و فارنهایت (قبل از تغییر)
- دو خط بعدی: دما به سلسیوس و فارنهایت (بعد از تغییر)

## مثال
```
ورودی:
0
100

خروجی:
سلسیوس: 0.00
فارنهایت: 32.00
سلسیوس: 37.78
فارنهایت: 100.00
```

## راهنمایی
- C = (F - 32) × 5/9
- F = (C × 9/5) + 32
