# سوال E0049 - Property برای زمان

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلاس Time با property برای محاسبه کل دقیقه‌ها ایجاد کند.

## ورودی
- خط اول: ساعت
- خط دوم: دقیقه

## خروجی
- زمان به فرمت HH:MM
- کل دقیقه‌ها

## مثال
```
ورودی:
14
30

خروجی:
زمان: 14:30
کل دقیقه‌ها: 870
```

## راهنمایی
- کل دقیقه‌ها = (ساعت × 60) + دقیقه
- ساعت باید بین 0 تا 23 باشد
- دقیقه باید بین 0 تا 59 باشد
