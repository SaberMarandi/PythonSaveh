# سوال G0001 - تست ساده با assert

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک تابع جمع دو عدد را تست کند با استفاده از assert.

## ورودی
دو عدد صحیح

## خروجی
نتیجه تست

## مثال
```
ورودی:
2, 3

خروجی:
Test passed!
```

## راهنمایی
- از assert برای بررسی نتیجه استفاده کنید
- تست‌های مختلف بنویسید
