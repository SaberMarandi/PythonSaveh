# سوال A0197 - محاسبه محیط دایره

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که الگوریتم Huffman Coding را برای فشرده‌سازی متن پیاده‌سازی کند.

## ورودی
یک رشته متن

## خروجی
- جدول کدهای هافمن
- متن کدگذاری شده
- نرخ فشرده‌سازی

## مثال
```
ورودی:
aabcccccaaa

خروجی:
a: 0
c: 10
b: 11
Encoded: 001011101010101000
Compression ratio: 54.5%
```

## راهنمایی
- فرکانس کاراکترها را محاسبه کنید
- درخت هافمن با priority queue بسازید
- کدهای باینری را از درخت استخراج کنید
- متن را با کدهای جدید رمز کنید
