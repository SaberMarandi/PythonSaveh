# سوال A0094 - جمع اعداد زوج

## صورت مسئله
برنامه‌ای بنویسید که تعدادی عدد دریافت کند و جمع اعداد زوج را محاسبه کند.

## ورودی
- خط اول: تعداد اعداد n
- خط دوم: n عدد صحیح (با فاصله)

## خروجی
جمع اعداد زوج

## مثال
```
ورودی:
5
3 7 2 9 4

خروجی:
6
```

## راهنمایی
- از حلقه for برای پیمایش اعداد استفاده کنید
- با عملگر % بررسی کنید که عدد زوج است
- اعداد زوج را جمع کنید
