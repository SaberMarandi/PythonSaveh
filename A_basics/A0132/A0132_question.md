# سوال A0132 - جستجوی دودویی

## صورت مسئله
برنامه‌ای بنویسید که با استفاده از جستجوی دودویی یک عدد را در لیست مرتب شده پیدا کند.

## ورودی
- خط اول: تعداد اعداد n
- خط دوم: n عدد صحیح مرتب شده (با فاصله)
- خط سوم: عدد جستجو

## خروجی
اندیس عدد (از 0 شروع) یا -1 اگر پیدا نشد

## مثال
```
ورودی:
5
2 3 4 7 9
7

خروجی:
3
```

## راهنمایی
- لیست باید مرتب شده باشد
- از الگوریتم جستجوی دودویی استفاده کنید
- در هر مرحله نیمی از لیست را حذف کنید
