# سوال A0085 - حذف فاصله‌ها

## صورت مسئله
برنامه‌ای بنویسید که یک رشته دریافت کند و تمام فاصله‌های آن را حذف کند.

## ورودی
یک رشته

## خروجی
رشته بدون فاصله

## مثال
```
ورودی:
Hello World Python

خروجی:
HelloWorldPython
```

## راهنمایی
- از متد replace() استفاده کنید
- یا از join() با split() استفاده کنید
