# سوال A0118 - مجموع مکعبات

## صورت مسئله
برنامه‌ای بنویسید که یک عدد n دریافت کند و مجموع مکعبات n عدد طبیعی اول را محاسبه کند.

## ورودی
یک عدد صحیح n

## خروجی
مجموع 1³ + 2³ + 3³ + ... + n³

## مثال
```
ورودی:
4

خروجی:
100
```
توضیح: 1 + 8 + 27 + 64 = 100

## راهنمایی
- از فرمول [n * (n + 1) / 2]² استفاده کنید
- یا از حلقه for استفاده کنید
