# سوال A0080 - چاپ الگوی اعداد مثلثی

## صورت مسئله
برنامه‌ای بنویسید که یک عدد صحیح n دریافت کند و یک مثلث از اعداد چاپ کند.

## ورودی
یک عدد صحیح n (تعداد سطرها)

## خروجی
مثلث اعداد با n سطر

## مثال
```
ورودی:
5

خروجی:
1
12
123
1234
12345
```

## راهنمایی
- از حلقه تو در تو استفاده کنید
- حلقه بیرونی برای سطرها
- حلقه درونی برای چاپ اعداد در هر سطر
