# سوال A0087 - اولین و آخرین کاراکتر

## صورت مسئله
برنامه‌ای بنویسید که یک رشته دریافت کند و اولین و آخرین کاراکتر آن را چاپ کند.

## ورودی
یک رشته

## خروجی
اولین و آخرین کاراکتر (با فاصله)

## مثال
```
ورودی:
Python

خروجی:
P n
```

## راهنمایی
- از اندیس‌گذاری استفاده کنید
- اولین کاراکتر: [0]
- آخرین کاراکتر: [-1]
