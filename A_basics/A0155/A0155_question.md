# سوال A0155 - قدر مطلق بدون abs

## سطح: خیلی پیشرفته

## صورت مسئله
برنامه‌ای بنویسید که قدر مطلق یک عدد را بدون استفاده از تابع abs محاسبه کند.

## ورودی
یک عدد صحیح

## خروجی
قدر مطلق عدد

## مثال
```
ورودی:
-15

خروجی:
15
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
