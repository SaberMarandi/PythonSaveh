# سوال A0131 - جستجوی خطی

## صورت مسئله
برنامه‌ای بنویسید که با استفاده از جستجوی خطی یک عدد را در لیست پیدا کند و اندیس آن را برگرداند.

## ورودی
- خط اول: تعداد اعداد n
- خط دوم: n عدد صحیح (با فاصله)
- خط سوم: عدد جستجو

## خروجی
اندیس عدد (از 0 شروع) یا -1 اگر پیدا نشد

## مثال
```
ورودی:
5
3 7 2 9 4
9

خروجی:
3
```

## راهنمایی
- از حلقه for برای پیمایش لیست استفاده کنید
- هر عنصر را با عدد جستجو مقایسه کنید
