# سوال A0129 - ترکیب (nCr)

## صورت مسئله
برنامه‌ای بنویسید که ترکیب n از r را محاسبه کند (nCr).

## ورودی
دو عدد صحیح n و r

## خروجی
nCr

## مثال
```
ورودی:
5 2

خروجی:
10
```

## راهنمایی
- از فرمول nCr = n! / (r! * (n-r)!) استفاده کنید
- یا از بازگشت استفاده کنید: C(n,r) = C(n-1,r-1) + C(n-1,r)
