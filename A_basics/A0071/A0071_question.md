# سوال A0071 - الگوی مثلث قائم‌الزاویه

## صورت مسئله
برنامه‌ای بنویسید که با دریافت عدد n، یک مثلث قائم‌الزاویه از ستاره با n سطر چاپ کند.

## ورودی
یک عدد صحیح n (1 ≤ n ≤ 20)

## خروجی
n سطر که در هر سطر i، تعداد i ستاره چاپ شود.

## مثال
```
ورودی:
5

خروجی:
*
**
***
****
*****
```

## راهنمایی
- از حلقه for استفاده کنید
- در هر تکرار، تعداد ستاره‌ها برابر با شماره سطر است
- می‌توانید از عملگر * برای تکرار رشته استفاده کنید
