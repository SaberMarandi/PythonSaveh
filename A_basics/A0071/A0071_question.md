# سوال A0071 - چاپ مثلث ستاره

## صورت مسئله
برنامه‌ای بنویسید که یک عدد صحیح n دریافت کند و یک مثلث از ستاره‌ها با n سطر چاپ کند.

## ورودی
یک عدد صحیح n (تعداد سطرها)

## خروجی
مثلث ستاره با n سطر

## مثال
```
ورودی:
5

خروجی:
*
**
***
****
*****
```

## راهنمایی
- از حلقه for استفاده کنید
- در هر سطر i، تعداد i ستاره چاپ کنید
- از عملگر * برای تکرار رشته استفاده کنید
