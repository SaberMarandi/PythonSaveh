# سوال A0111 - دنباله فیبوناچی تا n

## صورت مسئله
برنامه‌ای بنویسید که یک عدد n دریافت کند و n عدد اول دنباله فیبوناچی را چاپ کند.

## ورودی
یک عدد صحیح n

## خروجی
n عدد اول فیبوناچی (با فاصله)

## مثال
```
ورودی:
7

خروجی:
0 1 1 2 3 5 8
```

## راهنمایی
- دنباله فیبوناچی: 0, 1, 1, 2, 3, 5, 8, ...
- هر عدد مجموع دو عدد قبلی است
- از حلقه for استفاده کنید
