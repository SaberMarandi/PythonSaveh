# سوال A0123 - فیبوناچی با بازگشت

## صورت مسئله
برنامه‌ای بنویسید که با استفاده از بازگشت n-امین عدد فیبوناچی را محاسبه کند.

## ورودی
یک عدد صحیح n

## خروجی
n-امین عدد فیبوناچی

## مثال
```
ورودی:
7

خروجی:
8
```

## راهنمایی
- تابع بازگشتی بنویسید
- حالت پایه: fib(0) = 0, fib(1) = 1
- حالت بازگشتی: fib(n) = fib(n-1) + fib(n-2)
