# سوال A0098 - تعداد اعداد مثبت و منفی

## صورت مسئله
برنامه‌ای بنویسید که تعدادی عدد دریافت کند و تعداد اعداد مثبت و منفی را بشمارد.

## ورودی
- خط اول: تعداد اعداد n
- خط دوم: n عدد صحیح (با فاصله)

## خروجی
- خط اول: تعداد اعداد مثبت
- خط دوم: تعداد اعداد منفی

## مثال
```
ورودی:
6
3 -7 2 -9 4 0

خروجی:
3
2
```

## راهنمایی
- از حلقه for برای پیمایش اعداد استفاده کنید
- دو متغیر برای شمارش مثبت و منفی تعریف کنید
- عدد صفر را نادیده بگیرید
