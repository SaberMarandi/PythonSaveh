# سوال A0149 - محاسبه توان با الگوریتم سریع

## صورت مسئله
برنامه‌ای بنویسید که با استفاده از الگوریتم توان‌رسانی سریع (fast exponentiation) a به توان b را محاسبه کند.

## ورودی
دو عدد صحیح a و b

## خروجی
a به توان b

## مثال
```
ورودی:
2 10

خروجی:
1024
```

## راهنمایی
- از الگوریتم توان‌رسانی سریع استفاده کنید
- پیچیدگی زمانی: O(log b)
- اگر b زوج باشد: a^b = (a^2)^(b/2)
- اگر b فرد باشد: a^b = a × a^(b-1)
