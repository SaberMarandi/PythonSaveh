# سوال A0075 - چاپ هرم ستاره

## صورت مسئله
برنامه‌ای بنویسید که یک عدد صحیح n دریافت کند و یک هرم از ستاره‌ها با n سطر چاپ کند.

## ورودی
یک عدد صحیح n (تعداد سطرها)

## خروجی
هرم ستاره با n سطر

## مثال
```
ورودی:
5

خروجی:
    *
   ***
  *****
 *******
*********
```

## راهنمایی
- در هر سطر i، تعداد ستاره‌ها برابر (2*i - 1) است
- قبل از ستاره‌ها، (n - i) فاصله قرار دهید
- از حلقه for استفاده کنید
