# سوال A0075 - فاکتوریل

## صورت مسئله
برنامه‌ای بنویسید که فاکتوریل عدد n را محاسبه کند.

## ورودی
یک عدد صحیح n (0 ≤ n ≤ 20)

## خروجی
فاکتوریل n (n!)

## مثال
```
ورودی:
5

خروجی:
120
```

توضیح: 5! = 5 × 4 × 3 × 2 × 1 = 120

```
ورودی:
0

خروجی:
1
```

## راهنمایی
- فاکتوریل 0 برابر 1 است
- از حلقه for برای ضرب اعداد استفاده کنید
- یا می‌توانید از ماژول math.factorial استفاده کنید
