# سوال A0143 - پیدا کردن عنصر اکثریت

## صورت مسئله
برنامه‌ای بنویسید که عنصر اکثریت در لیست را پیدا کند (عنصری که بیش از n/2 بار تکرار شده). اگر چنین عنصری وجود نداشت -1 برگردانید.

## ورودی
- خط اول: تعداد اعداد n
- خط دوم: n عدد صحیح (با فاصله)

## خروجی
عنصر اکثریت یا -1

## مثال
```
ورودی:
7
3 3 4 2 3 3 3

خروجی:
3
```

## راهنمایی
- از الگوریتم Boyer-Moore Voting استفاده کنید
- یا فراوانی هر عنصر را بشمارید
