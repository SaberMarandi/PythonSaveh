# سوال A0177 - تبدیل به هگزادسیمال

## سطح: خیلی پیشرفته

## صورت مسئله
برنامه‌ای بنویسید که یک عدد دسیمال را به هگزادسیمال تبدیل کند.

## ورودی
یک عدد صحیح مثبت

## خروجی
معادل هگزادسیمال عدد

## مثال
```
ورودی:
255

خروجی:
FF
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
