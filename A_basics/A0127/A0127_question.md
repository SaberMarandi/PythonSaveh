# سوال A0127 - کوچکترین مضرب مشترک (LCM)

## صورت مسئله
برنامه‌ای بنویسید که کوچکترین مضرب مشترک دو عدد را محاسبه کند.

## ورودی
دو عدد صحیح a و b

## خروجی
LCM(a, b)

## مثال
```
ورودی:
12 18

خروجی:
36
```

## راهنمایی
- از رابطه LCM(a, b) = (a * b) / GCD(a, b) استفاده کنید
- ابتدا GCD را محاسبه کنید
