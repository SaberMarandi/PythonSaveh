# سوال A0112 - مجموع دنباله حسابی

## صورت مسئله
برنامه‌ای بنویسید که اولین جمله (a)، قدر نسبت (d) و تعداد جملات (n) یک دنباله حسابی را دریافت کند و مجموع n جمله اول را محاسبه کند.

## ورودی
سه عدد صحیح a، d، n

## خروجی
مجموع n جمله اول

## مثال
```
ورودی:
2 3 5

خروجی:
40
```
توضیح: 2 + 5 + 8 + 11 + 14 = 40

## راهنمایی
- دنباله حسابی: a, a+d, a+2d, ...
- از حلقه for برای محاسبه استفاده کنید
- یا از فرمول مجموع استفاده کنید: n/2 * (2a + (n-1)d)
