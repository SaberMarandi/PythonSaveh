# سوال A0072 - چاپ مثلث معکوس ستاره

## صورت مسئله
برنامه‌ای بنویسید که یک عدد صحیح n دریافت کند و یک مثلث معکوس از ستاره‌ها با n سطر چاپ کند.

## ورودی
یک عدد صحیح n (تعداد سطرها)

## خروجی
مثلث معکوس ستاره با n سطر

## مثال
```
ورودی:
5

خروجی:
*****
****
***
**
*
```

## راهنمایی
- از حلقه for استفاده کنید
- در هر سطر، تعداد ستاره‌ها کاهش می‌یابد
- می‌توانید از range با مقدار کاهشی استفاده کنید
