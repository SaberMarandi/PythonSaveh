# سوال A0187 - معکوس بیتها

## سطح: خیلی پیشرفته

## صورت مسئله
برنامه‌ای بنویسید که ترتیب بیت‌های یک عدد را معکوس کند.

## ورودی
یک عدد صحیح مثبت

## خروجی
عدد با بیت‌های معکوس

## مثال
```
ورودی:
12

خروجی:
3
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
