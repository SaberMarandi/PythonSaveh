# سوال A0117 - مجموع مربعات

## صورت مسئله
برنامه‌ای بنویسید که یک عدد n دریافت کند و مجموع مربعات n عدد طبیعی اول را محاسبه کند.

## ورودی
یک عدد صحیح n

## خروجی
مجموع 1² + 2² + 3² + ... + n²

## مثال
```
ورودی:
5

خروجی:
55
```
توضیح: 1 + 4 + 9 + 16 + 25 = 55

## راهنمایی
- از فرمول n * (n + 1) * (2n + 1) / 6 استفاده کنید
- یا از حلقه for استفاده کنید
