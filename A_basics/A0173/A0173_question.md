# سوال A0173 - کوچکترین رقم

## سطح: خیلی پیشرفته

## صورت مسئله
برنامه‌ای بنویسید که کوچکترین رقم یک عدد را پیدا کند.

## ورودی
یک عدد صحیح مثبت

## خروجی
کوچکترین رقم

## مثال
```
ورودی:
39472

خروجی:
2
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
