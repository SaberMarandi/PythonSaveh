# سوال A0073 - بررسی عدد اول

## صورت مسئله
برنامه‌ای بنویسید که بررسی کند عدد ورودی اول است یا خیر.

## ورودی
یک عدد صحیح n (2 ≤ n ≤ 10000)

## خروجی
اگر عدد اول است "Yes" و در غیر این صورت "No" چاپ کنید.

## مثال
```
ورودی:
7

خروجی:
Yes
```

```
ورودی:
12

خروجی:
No
```

## راهنمایی
- عدد اول فقط بر 1 و خودش بخش‌پذیر است
- برای بررسی، کافی است تا جذر n را چک کنید
- از حلقه for و شرط if استفاده کنید
