# سوال A0073 - چاپ مربع ستاره

## صورت مسئله
برنامه‌ای بنویسید که یک عدد صحیح n دریافت کند و یک مربع از ستاره‌ها با ابعاد n×n چاپ کند.

## ورودی
یک عدد صحیح n (ابعاد مربع)

## خروجی
مربع ستاره با ابعاد n×n

## مثال
```
ورودی:
4

خروجی:
****
****
****
****
```

## راهنمایی
- از حلقه for استفاده کنید
- در هر سطر، n ستاره چاپ کنید
- می‌توانید از تکرار رشته استفاده کنید
