# سوال A0009 - محاسبه محیط مستطیل

## صورت مسئله
برنامه‌ای بنویسید که طول و عرض یک مستطیل را از کاربر بگیرد و محیط آن را محاسبه کند.

## ورودی
دو عدد صحیح (طول و عرض)

## خروجی
محیط مستطیل

## مثال
```
ورودی:
5
8

خروجی:
26
```

## راهنمایی
- محیط مستطیل = 2 × (طول + عرض)
- یا می‌توانید از فرمول: 2 × طول + 2 × عرض استفاده کنید
