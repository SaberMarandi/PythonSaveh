# سوال A0190 - فیبوناچی با برنامه‌نویسی پویا

## سطح: متوسط

## صورت مسئله
برنامه‌ای بنویسید که اعداد فیبوناچی تا n امین جمله را با استفاده از برنامه‌نویسی پویا محاسبه کند.

## ورودی
یک عدد صحیح n

## خروجی
n امین عدد فیبوناچی

## مثال
```
ورودی:
10

خروجی:
55
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
