# سوال A0088 - تکرار هر کاراکتر

## صورت مسئله
برنامه‌ای بنویسید که یک رشته و یک عدد n دریافت کند و هر کاراکتر را n بار تکرار کند.

## ورودی
- خط اول: یک رشته
- خط دوم: یک عدد صحیح n

## خروجی
رشته با کاراکترهای تکرار شده

## مثال
```
ورودی:
abc
3

خروجی:
aaabbbccc
```

## راهنمایی
- از حلقه for برای پیمایش کاراکترها استفاده کنید
- هر کاراکتر را n بار تکرار کنید
