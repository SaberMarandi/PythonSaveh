# سوال A0097 - حذف تکراری‌ها

## صورت مسئله
برنامه‌ای بنویسید که تعدادی عدد دریافت کند و اعداد تکراری را حذف کند (فقط یک بار هر عدد را نمایش دهد).

## ورودی
- خط اول: تعداد اعداد n
- خط دوم: n عدد صحیح (با فاصله)

## خروجی
اعداد بدون تکرار (با فاصله، به ترتیب ظهور)

## مثال
```
ورودی:
7
3 7 2 7 4 3 9

خروجی:
3 7 2 4 9
```

## راهنمایی
- از set برای حذف تکراری‌ها استفاده کنید
- یا از dict.fromkeys() برای حفظ ترتیب استفاده کنید
