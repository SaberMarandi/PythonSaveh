# سوال A0090 - شمارش یک کاراکتر

## صورت مسئله
برنامه‌ای بنویسید که یک رشته و یک کاراکتر دریافت کند و تعداد تکرار آن کاراکتر را در رشته بشمارد.

## ورودی
- خط اول: یک رشته
- خط دوم: یک کاراکتر

## خروجی
تعداد تکرار کاراکتر

## مثال
```
ورودی:
Hello World
o

خروجی:
2
```

## راهنمایی
- از متد count() استفاده کنید
- یا از حلقه for برای شمارش استفاده کنید
