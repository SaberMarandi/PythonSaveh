# سوال A0082 - شمارش حروف صدادار

## صورت مسئله
برنامه‌ای بنویسید که یک رشته دریافت کند و تعداد حروف صدادار (a, e, i, o, u) را در آن بشمارد.

## ورودی
یک رشته

## خروجی
تعداد حروف صدادار

## مثال
```
ورودی:
Hello World

خروجی:
3
```

## راهنمایی
- حروف صدادار: a, e, i, o, u
- از حلقه for برای بررسی هر کاراکتر استفاده کنید
- از متد lower() برای تبدیل به حروف کوچک استفاده کنید
