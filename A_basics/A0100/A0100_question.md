# سوال A0100 - جستجوی عدد در لیست

## صورت مسئله
برنامه‌ای بنویسید که تعدادی عدد و یک عدد جستجو دریافت کند و بررسی کند آیا عدد جستجو در لیست وجود دارد یا خیر.

## ورودی
- خط اول: تعداد اعداد n
- خط دوم: n عدد صحیح (با فاصله)
- خط سوم: عدد جستجو

## خروجی
YES اگر عدد وجود داشته باشد، در غیر این صورت NO

## مثال
```
ورودی:
5
3 7 2 9 4
7

خروجی:
YES
```

## راهنمایی
- از عملگر in برای بررسی وجود استفاده کنید
- یا از متد count() استفاده کنید
