# سوال A0157 - جذر بدون sqrt

## سطح: خیلی پیشرفته

## صورت مسئله
برنامه‌ای بنویسید که جذر یک عدد را بدون استفاده از تابع sqrt محاسبه کند.

## ورودی
یک عدد مثبت

## خروجی
جذر عدد (صحیح)

## مثال
```
ورودی:
16

خروجی:
4
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
