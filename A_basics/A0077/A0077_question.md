# سوال A0077 - اعداد فیبوناچی

## صورت مسئله
برنامه‌ای بنویسید که n عدد اول دنباله فیبوناچی را چاپ کند.

دنباله فیبوناچی: 0, 1, 1, 2, 3, 5, 8, 13, ...

## ورودی
یک عدد صحیح n (1 ≤ n ≤ 30)

## خروجی
n عدد اول فیبوناچی در یک خط با فاصله

## مثال
```
ورودی:
7

خروجی:
0 1 1 2 3 5 8
```

## راهنمایی
- دو عدد اول: 0 و 1
- هر عدد بعدی = مجموع دو عدد قبلی
- از دو متغیر برای نگهداری دو عدد قبلی استفاده کنید
