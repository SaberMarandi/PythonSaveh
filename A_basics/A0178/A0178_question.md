# سوال A0178 - تبدیل به اکتال

## سطح: خیلی پیشرفته

## صورت مسئله
برنامه‌ای بنویسید که یک عدد دسیمال را به اکتال (مبنای 8) تبدیل کند.

## ورودی
یک عدد صحیح مثبت

## خروجی
معادل اکتال عدد

## مثال
```
ورودی:
64

خروجی:
100
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
