# سوال A0076 - چاپ هرم معکوس ستاره

## صورت مسئله
برنامه‌ای بنویسید که یک عدد صحیح n دریافت کند و یک هرم معکوس از ستاره‌ها با n سطر چاپ کند.

## ورودی
یک عدد صحیح n (تعداد سطرها)

## خروجی
هرم معکوس ستاره با n سطر

## مثال
```
ورودی:
5

خروجی:
*********
 *******
  *****
   ***
    *
```

## راهنمایی
- در هر سطر i، تعداد ستاره‌ها کاهش می‌یابد
- قبل از ستاره‌ها، فاصله‌ها افزایش می‌یابند
- از حلقه for با مقدار کاهشی استفاده کنید
