# سوال A0076 - الگوی الماس

## صورت مسئله
برنامه‌ای بنویسید که یک الگوی الماس از ستاره با n سطر (برای نیمه بالا) چاپ کند.

## ورودی
یک عدد صحیح n (1 ≤ n ≤ 10)

## خروجی
الگوی الماس

## مثال
```
ورودی:
3

خروجی:
  *
 ***
*****
 ***
  *
```

## راهنمایی
- الماس از دو مثلث تشکیل شده است
- ابتدا مثلث صعودی و سپس نزولی
- از فاصله برای تراز کردن استفاده کنید
