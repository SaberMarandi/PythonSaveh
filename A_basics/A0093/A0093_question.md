# سوال A0093 - مرتب‌سازی اعداد

## صورت مسئله
برنامه‌ای بنویسید که تعدادی عدد دریافت کند و آنها را به ترتیب صعودی مرتب کند.

## ورودی
- خط اول: تعداد اعداد n
- خط دوم: n عدد صحیح (با فاصله)

## خروجی
اعداد مرتب شده (با فاصله)

## مثال
```
ورودی:
5
3 7 2 9 4

خروجی:
2 3 4 7 9
```

## راهنمایی
- از تابع sorted() یا متد sort() استفاده کنید
- برای چاپ از join() استفاده کنید
