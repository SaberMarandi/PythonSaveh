# سوال A0146 - طول طولانی‌ترین زیررشته بدون تکرار

## صورت مسئله
برنامه‌ای بنویسید که طول طولانی‌ترین زیررشته بدون کاراکتر تکراری را پیدا کند.

## ورودی
یک رشته

## خروجی
طول طولانی‌ترین زیررشته بدون تکرار

## مثال
```
ورودی:
abcabcbb

خروجی:
3
```
توضیح: "abc" طولانی‌ترین زیررشته بدون تکرار است

## راهنمایی
- از الگوریتم پنجره کشویی (sliding window) استفاده کنید
- از set برای نگهداری کاراکترهای منحصر به فرد استفاده کنید
