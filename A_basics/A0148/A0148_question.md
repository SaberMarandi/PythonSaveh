# سوال A0148 - تبدیل عدد به رومی

## صورت مسئله
برنامه‌ای بنویسید که یک عدد صحیح را به عدد رومی تبدیل کند.

## ورودی
یک عدد صحیح (1 تا 3999)

## خروجی
عدد رومی معادل

## مثال
```
ورودی:
1994

خروجی:
MCMXCIV
```

## راهنمایی
- از دیکشنری برای نگهداری مقادیر رومی استفاده کنید
- از بزرگترین به کوچکترین پیش بروید
