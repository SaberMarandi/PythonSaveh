# سوال A0124 - توان با بازگشت

## صورت مسئله
برنامه‌ای بنویسید که با استفاده از بازگشت a به توان b را محاسبه کند.

## ورودی
دو عدد صحیح a و b

## خروجی
a به توان b

## مثال
```
ورودی:
2 10

خروجی:
1024
```

## راهنمایی
- تابع بازگشتی بنویسید
- حالت پایه: power(a, 0) = 1
- حالت بازگشتی: power(a, b) = a * power(a, b-1)
