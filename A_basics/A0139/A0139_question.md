# سوال A0139 - چرخش آرایه

## صورت مسئله
برنامه‌ای بنویسید که یک آرایه را k بار به راست بچرخاند.

## ورودی
- خط اول: تعداد اعداد n
- خط دوم: n عدد صحیح (با فاصله)
- خط سوم: تعداد چرخش k

## خروجی
آرایه چرخانده شده (با فاصله)

## مثال
```
ورودی:
5
1 2 3 4 5
2

خروجی:
4 5 1 2 3
```

## راهنمایی
- از slicing استفاده کنید
- یا از الگوریتم معکوس کردن استفاده کنید
