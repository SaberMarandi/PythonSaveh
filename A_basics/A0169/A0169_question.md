# سوال A0169 - معکوس عدد

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که مسئله Edit Distance (Levenshtein Distance) را حل کند.

## ورودی
دو رشته در دو خط

## خروجی
کمترین تعداد عملیات برای تبدیل رشته اول به دوم

## مثال
```
ورودی:
kitten
sitting

خروجی:
3
```
توضیح: kitten → sitten → sittin → sitting

## راهنمایی
- سه عملیات: insert، delete، replace
- dp[i][j] = edit distance بین i کاراکتر اول رشته اول و j کاراکتر اول رشته دوم
- اگر کاراکترها برابر باشند: dp[i][j] = dp[i-1][j-1]
- در غیر این صورت: dp[i][j] = 1 + min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1])
