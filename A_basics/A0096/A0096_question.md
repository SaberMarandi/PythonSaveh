# سوال A0096 - میانگین اعداد

## صورت مسئله
برنامه‌ای بنویسید که تعدادی عدد دریافت کند و میانگین آنها را محاسبه کند.

## ورودی
- خط اول: تعداد اعداد n
- خط دوم: n عدد صحیح (با فاصله)

## خروجی
میانگین اعداد (با دو رقم اعشار)

## مثال
```
ورودی:
5
3 7 2 9 4

خروجی:
5.00
```

## راهنمایی
- جمع اعداد را بر تعداد آنها تقسیم کنید
- از تابع sum() برای جمع استفاده کنید
- از round() برای گرد کردن استفاده کنید
