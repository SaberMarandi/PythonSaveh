# سوال A0162 - بررسی عدد اول با بهینه‌سازی

## سطح: خیلی پیشرفته

## صورت مسئله
برنامه‌ای بنویسید که با الگوریتم بهینه بررسی کند عدد اول است یا نه.

## ورودی
یک عدد صحیح مثبت

## خروجی
اول یا غیر اول

## مثال
```
ورودی:
97

خروجی:
اول
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
