# سوال A0154 - بررسی عدد مثبت یا منفی بدون مقایسه

## سطح: خیلی پیشرفته

## صورت مسئله
برنامه‌ای بنویسید که بدون استفاده از عملگرهای مقایسه بررسی کند عدد مثبت است یا منفی.

## ورودی
یک عدد صحیح

## خروجی
مثبت، منفی یا صفر

## مثال
```
ورودی:
-5

خروجی:
منفی
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
