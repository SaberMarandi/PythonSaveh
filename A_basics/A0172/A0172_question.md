# سوال A0172 - بزرگترین رقم

## سطح: خیلی پیشرفته

## صورت مسئله
برنامه‌ای بنویسید که بزرگترین رقم یک عدد را پیدا کند.

## ورودی
یک عدد صحیح مثبت

## خروجی
بزرگترین رقم

## مثال
```
ورودی:
39472

خروجی:
9
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
