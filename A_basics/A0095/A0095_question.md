# سوال A0095 - جمع اعداد فرد

## صورت مسئله
برنامه‌ای بنویسید که تعدادی عدد دریافت کند و جمع اعداد فرد را محاسبه کند.

## ورودی
- خط اول: تعداد اعداد n
- خط دوم: n عدد صحیح (با فاصله)

## خروجی
جمع اعداد فرد

## مثال
```
ورودی:
5
3 7 2 9 4

خروجی:
19
```

## راهنمایی
- از حلقه for برای پیمایش اعداد استفاده کنید
- با عملگر % بررسی کنید که عدد فرد است
- اعداد فرد را جمع کنید
