# سوال A0188 - بررسی توان 2

## سطح: خیلی پیشرفته

## صورت مسئله
برنامه‌ای بنویسید که بررسی کند آیا یک عدد توان 2 است یا خیر.

## ورودی
یک عدد صحیح مثبت

## خروجی
YES اگر توان 2 باشد، در غیر این صورت NO

## مثال
```
ورودی:
8

خروجی:
YES
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
