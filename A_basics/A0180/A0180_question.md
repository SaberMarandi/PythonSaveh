# سوال A0180 - بررسی آناگرام

## سطح: متوسط

## صورت مسئله
برنامه‌ای بنویسید که بررسی کند آیا دو رشته anagram یکدیگر هستند یا خیر.

## ورودی
دو رشته در دو خط

## خروجی
YES اگر anagram باشند، در غیر این صورت NO

## مثال
```
ورودی:
listen
silent

خروجی:
YES
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
