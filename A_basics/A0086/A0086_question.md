# سوال A0086 - تبدیل به حروف بزرگ و کوچک

## صورت مسئله
برنامه‌ای بنویسید که یک رشته دریافت کند و آن را به صورت متناوب به حروف بزرگ و کوچک تبدیل کند.

## ورودی
یک رشته

## خروجی
رشته با حروف متناوب بزرگ و کوچک

## مثال
```
ورودی:
hello

خروجی:
HeLlO
```

## راهنمایی
- از حلقه for با enumerate استفاده کنید
- برای اندیس‌های زوج از upper() و برای فرد از lower() استفاده کنید
