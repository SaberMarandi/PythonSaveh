# سوال A0151 - تبدیل رشته به عدد

## سطح: خیلی پیشرفته

## صورت مسئله
برنامه‌ای بنویسید که یک رشته عددی را به عدد صحیح تبدیل کند (بدون استفاده از int()).

## ورودی
یک رشته عددی

## خروجی
معادل عددی رشته

## مثال
```
ورودی:
12345

خروجی:
12345
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
