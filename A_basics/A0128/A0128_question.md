# سوال A0128 - برج هانوی

## صورت مسئله
برنامه‌ای بنویسید که تعداد حرکات لازم برای حل مسئله برج هانوی با n دیسک را محاسبه کند.

## ورودی
یک عدد صحیح n (تعداد دیسک‌ها)

## خروجی
تعداد حرکات لازم

## مثال
```
ورودی:
3

خروجی:
7
```

## راهنمایی
- تعداد حرکات برای n دیسک: 2^n - 1
- یا با بازگشت: hanoi(n) = 2 * hanoi(n-1) + 1
