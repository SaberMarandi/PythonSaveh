# سوال A0166 - سری فیبوناچی

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که الگوریتم Counting Sort را پیاده‌سازی کند.

## ورودی
- خط اول: تعداد عناصر
- خط دوم: عناصر آرایه (اعداد غیرمنفی کوچک)

## خروجی
آرایه مرتب‌شده

## مثال
```
ورودی:
8
4 2 2 8 3 3 1 1

خروجی:
1 1 2 2 3 3 4 8
```

## راهنمایی
- آرایه count برای شمارش فرکانس هر عدد
- بیشترین مقدار را پیدا کنید
- فرکانس هر عدد را شمارش کنید
- آرایه خروجی را بر اساس فرکانس‌ها بسازید
- پیچیدگی زمانی O(n + k)
