# سوال A0163 - فاکتوریل بزرگ

## سطح: خیلی پیشرفته

## صورت مسئله
برنامه‌ای بنویسید که فاکتوریل اعداد بزرگ را محاسبه کند.

## ورودی
یک عدد صحیح مثبت

## خروجی
فاکتوریل عدد

## مثال
```
ورودی:
20

خروجی:
2432902008176640000
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
