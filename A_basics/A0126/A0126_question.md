# سوال A0126 - بزرگترین مقسوم‌علیه مشترک (GCD)

## صورت مسئله
برنامه‌ای بنویسید که با استفاده از الگوریتم اقلیدس بزرگترین مقسوم‌علیه مشترک دو عدد را محاسبه کند.

## ورودی
دو عدد صحیح a و b

## خروجی
GCD(a, b)

## مثال
```
ورودی:
48 18

خروجی:
6
```

## راهنمایی
- از الگوریتم اقلیدس استفاده کنید
- GCD(a, b) = GCD(b, a mod b)
- حالت پایه: GCD(a, 0) = a
