# سوال A0103 - مجموع ارقام عدد

## صورت مسئله
برنامه‌ای بنویسید که یک عدد صحیح دریافت کند و مجموع ارقام آن را محاسبه کند.

## ورودی
یک عدد صحیح

## خروجی
مجموع ارقام عدد

## مثال
```
ورودی:
12345

خروجی:
15
```

## راهنمایی
- عدد را به رشته تبدیل کنید
- هر رقم را به عدد تبدیل کرده و جمع کنید
- یا با عملگر % و // ارقام را استخراج کنید
