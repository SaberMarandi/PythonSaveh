# سوال A0079 - چاپ الگوی شطرنجی

## صورت مسئله
برنامه‌ای بنویسید که یک عدد صحیح n دریافت کند و یک الگوی شطرنجی با ابعاد n×n چاپ کند (با استفاده از * و فاصه).

## ورودی
یک عدد صحیح n (ابعاد الگو)

## خروجی
الگوی شطرنجی با ابعاد n×n

## مثال
```
ورودی:
5

خروجی:
* * *
 * * 
* * *
 * * 
* * *
```

## راهنمایی
- در سطرهای فرد، با ستاره شروع کنید
- در سطرهای زوج، با فاصله شروع کنید
- از شرط if و عملگر % استفاده کنید
