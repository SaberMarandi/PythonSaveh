# سوال A0079 - تعداد ارقام

## صورت مسئله
برنامه‌ای بنویسید که تعداد ارقام یک عدد را بشمارد.

## ورودی
یک عدد صحیح

## خروجی
تعداد ارقام

## مثال
```
ورودی:
12345

خروجی:
5
```

```
ورودی:
7

خروجی:
1
```

## راهنمایی
- عدد را به رشته تبدیل کنید
- طول رشته = تعداد ارقام
- یا با تقسیم بر 10 تا صفر شدن
