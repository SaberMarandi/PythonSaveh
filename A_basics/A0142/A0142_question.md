# سوال A0142 - جابجایی دو عدد بدون متغیر سوم

## صورت مسئله
برنامه‌ای بنویسید که دو عدد را بدون استفاده از متغیر سوم جابجا کند.

## ورودی
دو عدد صحیح a و b

## خروجی
- خط اول: b
- خط دوم: a

## مثال
```
ورودی:
5 10

خروجی:
10
5
```

## راهنمایی
- از عملگرهای ریاضی استفاده کنید
- یا از tuple unpacking در پایتون استفاده کنید
