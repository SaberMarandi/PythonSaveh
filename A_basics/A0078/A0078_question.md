# سوال A0078 - چاپ مربع توخالی

## صورت مسئله
برنامه‌ای بنویسید که یک عدد صحیح n دریافت کند و یک مربع توخالی از ستاره‌ها با ابعاد n×n چاپ کند.

## ورودی
یک عدد صحیح n (ابعاد مربع)

## خروجی
مربع توخالی ستاره با ابعاد n×n

## مثال
```
ورودی:
5

خروجی:
*****
*   *
*   *
*   *
*****
```

## راهنمایی
- سطر اول و آخر پر از ستاره است
- سطرهای میانی فقط در ابتدا و انتها ستاره دارند
- از شرط if برای تشخیص سطر اول، آخر و میانی استفاده کنید
