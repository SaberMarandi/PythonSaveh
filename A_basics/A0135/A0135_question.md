# سوال A0135 - مرتب‌سازی درجی

## صورت مسئله
برنامه‌ای بنویسید که با استفاده از الگوریتم مرتب‌سازی درجی یک لیست را مرتب کند.

## ورودی
- خط اول: تعداد اعداد n
- خط دوم: n عدد صحیح (با فاصله)

## خروجی
لیست مرتب شده (با فاصله)

## مثال
```
ورودی:
5
3 7 2 9 4

خروجی:
2 3 4 7 9
```

## راهنمایی
- هر عنصر را در جای مناسب خود در بخش مرتب شده قرار دهید
- مانند مرتب کردن کارت‌های بازی
