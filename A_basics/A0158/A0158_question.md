# سوال A0158 - گرد کردن بدون round

## سطح: خیلی پیشرفته

## صورت مسئله
برنامه‌ای بنویسید که یک عدد اعشاری را بدون استفاده از تابع round گرد کند.

## ورودی
یک عدد اعشاری

## خروجی
عدد گرد شده

## مثال
```
ورودی:
3.7

خروجی:
4
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
