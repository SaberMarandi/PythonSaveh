# سوال A0140 - حذف عناصر تکراری از لیست مرتب

## صورت مسئله
برنامه‌ای بنویسید که عناصر تکراری را از یک لیست مرتب شده حذف کند.

## ورودی
- خط اول: تعداد اعداد n
- خط دوم: n عدد صحیح مرتب شده (با فاصله)

## خروجی
لیست بدون تکرار (با فاصله)

## مثال
```
ورودی:
8
1 1 2 2 2 3 4 4

خروجی:
1 2 3 4
```

## راهنمایی
- از یک حلقه استفاده کنید
- فقط عناصری را اضافه کنید که با عنصر قبلی متفاوت هستند
