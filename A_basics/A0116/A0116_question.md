# سوال A0116 - مجموع اعداد طبیعی

## صورت مسئله
برنامه‌ای بنویسید که یک عدد n دریافت کند و مجموع n عدد طبیعی اول را محاسبه کند.

## ورودی
یک عدد صحیح n

## خروجی
مجموع 1 + 2 + 3 + ... + n

## مثال
```
ورودی:
10

خروجی:
55
```

## راهنمایی
- از فرمول n * (n + 1) / 2 استفاده کنید
- یا از حلقه for استفاده کنید
