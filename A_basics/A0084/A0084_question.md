# سوال A0084 - شمارش کلمات

## صورت مسئله
برنامه‌ای بنویسید که یک جمله دریافت کند و تعداد کلمات آن را بشمارد.

## ورودی
یک جمله

## خروجی
تعداد کلمات

## مثال
```
ورودی:
Hello World Python

خروجی:
3
```

## راهنمایی
- از متد split() برای جدا کردن کلمات استفاده کنید
- از تابع len() برای شمارش استفاده کنید
