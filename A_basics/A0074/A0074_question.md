# سوال A0074 - معکوس عدد

## صورت مسئله
برنامه‌ای بنویسید که عدد ورودی را معکوس کند.

## ورودی
یک عدد صحیح مثبت

## خروجی
عدد معکوس شده

## مثال
```
ورودی:
12345

خروجی:
54321
```

```
ورودی:
100

خروجی:
1
```

## راهنمایی
- می‌توانید عدد را به رشته تبدیل کنید
- از slicing برای معکوس کردن رشته استفاده کنید
- یا با عملیات ریاضی (باقیمانده و تقسیم) معکوس کنید
