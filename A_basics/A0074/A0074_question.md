# سوال A0074 - چاپ مثلث راست‌گوشه

## صورت مسئله
برنامه‌ای بنویسید که یک عدد صحیح n دریافت کند و یک مثلث راست‌گوشه از ستاره‌ها با n سطر چاپ کند (با فاصله‌های مناسب).

## ورودی
یک عدد صحیح n (تعداد سطرها)

## خروجی
مثلث راست‌گوشه ستاره با n سطر

## مثال
```
ورودی:
5

خروجی:
    *
   **
  ***
 ****
*****
```

## راهنمایی
- از حلقه for استفاده کنید
- در هر سطر i، ابتدا (n-i) فاصله و سپس i ستاره چاپ کنید
- می‌توانید از تکرار رشته برای فاصله و ستاره استفاده کنید
