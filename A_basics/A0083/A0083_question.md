# سوال A0083 - بررسی پالیندروم

## صورت مسئله
برنامه‌ای بنویسید که یک رشته دریافت کند و بررسی کند آیا پالیندروم است یا خیر (از چپ و راست یکسان خوانده می‌شود).

## ورودی
یک رشته

## خروجی
YES اگر پالیندروم باشد، در غیر این صورت NO

## مثال
```
ورودی:
radar

خروجی:
YES
```

## راهنمایی
- رشته را با معکوس آن مقایسه کنید
- از slicing [::-1] برای معکوس کردن استفاده کنید
