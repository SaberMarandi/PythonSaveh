# سوال A0174 - مرتب‌سازی ارقام

## سطح: خیلی پیشرفته

## صورت مسئله
برنامه‌ای بنویسید که ارقام یک عدد را به صورت صعودی مرتب کند.

## ورودی
یک عدد صحیح مثبت

## خروجی
عدد با ارقام مرتب شده

## مثال
```
ورودی:
54321

خروجی:
12345
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
