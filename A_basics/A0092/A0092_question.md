# سوال A0092 - کوچکترین عدد در لیست

## صورت مسئله
برنامه‌ای بنویسید که تعدادی عدد دریافت کند و کوچکترین آنها را پیدا کند.

## ورودی
- خط اول: تعداد اعداد n
- خط دوم: n عدد صحیح (با فاصله)

## خروجی
کوچکترین عدد

## مثال
```
ورودی:
5
3 7 2 9 4

خروجی:
2
```

## راهنمایی
- از تابع min() استفاده کنید
- یا با حلقه for کوچکترین را پیدا کنید
