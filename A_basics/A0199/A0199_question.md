# سوال A0199 - محاسبه مساحت مربع

## سطح: خیلی پیشرفته

## صورت مسئله
برنامه‌ای بنویسید که مساحت مربع را با استفاده از طول ضلع محاسبه کند.

## ورودی
طول ضلع مربع

## خروجی
مساحت مربع

## مثال
```
ورودی:
5

خروجی:
25
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
