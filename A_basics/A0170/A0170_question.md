# سوال A0170 - مجموع ارقام عدد

## سطح: خیلی پیشرفته

## صورت مسئله
برنامه‌ای بنویسید که مجموع ارقام یک عدد را محاسبه کند.

## ورودی
یک عدد صحیح مثبت

## خروجی
مجموع ارقام

## مثال
```
ورودی:
1234

خروجی:
10
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
