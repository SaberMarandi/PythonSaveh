# سوال A0159 - باقیمانده بدون modulo

## سطح: خیلی پیشرفته

## صورت مسئله
برنامه‌ای بنویسید که باقیمانده تقسیم را بدون استفاده از عملگر % محاسبه کند.

## ورودی
دو عدد صحیح

## خروجی
باقیمانده تقسیم

## مثال
```
ورودی:
17
5

خروجی:
2
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
