# سوال A0175 - حذف صفرهای ابتدایی

## سطح: متوسط

## صورت مسئله
برنامه‌ای بنویسید که صفرهای ابتدایی یک رشته عددی را حذف کند.

## ورودی
یک رشته عددی

## خروجی
رشته بدون صفرهای ابتدایی

## مثال
```
ورودی:
00012300

خروجی:
12300
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
