# سوال A0113 - مجموع دنباله هندسی

## صورت مسئله
برنامه‌ای بنویسید که اولین جمله (a)، قدر نسبت (r) و تعداد جملات (n) یک دنباله هندسی را دریافت کند و مجموع n جمله اول را محاسبه کند.

## ورودی
سه عدد صحیح a، r، n

## خروجی
مجموع n جمله اول

## مثال
```
ورودی:
2 3 4

خروجی:
80
```
توضیح: 2 + 6 + 18 + 54 = 80

## راهنمایی
- دنباله هندسی: a, a*r, a*r², ...
- از حلقه for برای محاسبه استفاده کنید
- یا از فرمول مجموع استفاده کنید: a * (r^n - 1) / (r - 1)
