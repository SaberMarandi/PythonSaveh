# سوال A0122 - فاکتوریل با بازگشت

## صورت مسئله
برنامه‌ای بنویسید که با استفاده از بازگشت (recursion) فاکتوریل یک عدد را محاسبه کند.

## ورودی
یک عدد صحیح n

## خروجی
فاکتوریل n

## مثال
```
ورودی:
5

خروجی:
120
```

## راهنمایی
- تابع بازگشتی بنویسید
- حالت پایه: factorial(0) = 1
- حالت بازگشتی: factorial(n) = n * factorial(n-1)
