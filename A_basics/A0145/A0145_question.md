# سوال A0145 - بررسی آناگرام

## صورت مسئله
برنامه‌ای بنویسید که بررسی کند آیا دو رشته آناگرام یکدیگر هستند یا خیر. (آناگرام: دو رشته که از حروف یکسانی تشکیل شده‌اند)

## ورودی
- خط اول: رشته اول
- خط دوم: رشته دوم

## خروجی
YES اگر آناگرام باشند، در غیر این صورت NO

## مثال
```
ورودی:
listen
silent

خروجی:
YES
```

## راهنمایی
- دو رشته را مرتب کنید و مقایسه کنید
- یا فراوانی هر کاراکتر را بشمارید
