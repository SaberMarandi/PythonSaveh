# سوال A0138 - تعداد معکوس‌ها

## صورت مسئله
برنامه‌ای بنویسید که تعداد معکوس‌ها (inversions) در یک آرایه را بشمارد. معکوس زمانی است که i < j ولی arr[i] > arr[j].

## ورودی
- خط اول: تعداد اعداد n
- خط دوم: n عدد صحیح (با فاصله)

## خروجی
تعداد معکوس‌ها

## مثال
```
ورودی:
5
2 4 1 3 5

خروجی:
3
```
توضیح: (2,1), (4,1), (4,3)

## راهنمایی
- از دو حلقه تو در تو استفاده کنید
- برای هر جفت (i, j) که i < j بررسی کنید
