# سوال A0089 - جایگزینی کاراکتر

## صورت مسئله
برنامه‌ای بنویسید که یک رشته و دو کاراکتر دریافت کند و کاراکتر اول را با کاراکتر دوم جایگزین کند.

## ورودی
- خط اول: یک رشته
- خط دوم: کاراکتر قدیمی
- خط سوم: کاراکتر جدید

## خروجی
رشته با کاراکترهای جایگزین شده

## مثال
```
ورودی:
Hello World
o
a

خروجی:
Hella Warld
```

## راهنمایی
- از متد replace() استفاده کنید
