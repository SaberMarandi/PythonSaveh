# سوال A0144 - پیدا کردن عنصر گمشده

## صورت مسئله
برنامه‌ای بنویسید که عدد گمشده در دنباله 1 تا n را پیدا کند. (یک لیست با n-1 عدد داده می‌شود که شامل اعداد 1 تا n است به جز یک عدد)

## ورودی
- خط اول: n
- خط دوم: n-1 عدد صحیح (با فاصله)

## خروجی
عدد گمشده

## مثال
```
ورودی:
5
1 2 4 5

خروجی:
3
```

## راهنمایی
- مجموع اعداد 1 تا n را محاسبه کنید
- مجموع اعداد موجود را محاسبه کنید
- تفاوت آنها عدد گمشده است
