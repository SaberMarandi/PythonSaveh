# سوال A0125 - مجموع ارقام با بازگشت

## صورت مسئله
برنامه‌ای بنویسید که با استفاده از بازگشت مجموع ارقام یک عدد را محاسبه کند.

## ورودی
یک عدد صحیح

## خروجی
مجموع ارقام

## مثال
```
ورودی:
12345

خروجی:
15
```

## راهنمایی
- تابع بازگشتی بنویسید
- حالت پایه: اگر عدد کمتر از 10 باشد، خودش را برگردانید
- حالت بازگشتی: آخرین رقم + مجموع بقیه ارقام
