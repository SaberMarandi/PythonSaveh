# سوال A0184 - عملیات بیتی NOT

## سطح: خیلی پیشرفته

## صورت مسئله
برنامه‌ای بنویسید که عملیات بیتی NOT روی یک عدد انجام دهد.

## ورودی
یک عدد صحیح

## خروجی
نتیجه عملیات NOT بیتی

## مثال
```
ورودی:
5

خروجی:
-6
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
