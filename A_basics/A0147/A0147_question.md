# سوال A0147 - تبدیل رومی به عدد

## صورت مسئله
برنامه‌ای بنویسید که یک عدد رومی را به عدد صحیح تبدیل کند.

## ورودی
یک رشته (عدد رومی)

## خروجی
عدد صحیح معادل

## مثال
```
ورودی:
MCMXCIV

خروجی:
1994
```

## راهنمایی
- از دیکشنری برای نگهداری مقادیر رومی استفاده کنید
- اگر عدد بعدی بزرگتر باشد، کم کنید، در غیر این صورت اضافه کنید
