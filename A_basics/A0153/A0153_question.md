# سوال A0153 - بررسی عدد زوج یا فرد بدون modulo

## سطح: خیلی پیشرفته

## صورت مسئله
برنامه‌ای بنویسید که بدون استفاده از عملگر % بررسی کند عدد زوج است یا فرد.

## ورودی
یک عدد صحیح

## خروجی
زوج یا فرد

## مثال
```
ورودی:
7

خروجی:
فرد
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
