# سوال A0152 - تبدیل عدد به رشته

## سطح: خیلی پیشرفته

## صورت مسئله
برنامه‌ای بنویسید که یک عدد صحیح را به رشته تبدیل کند (بدون استفاده از str()).

## ورودی
یک عدد صحیح

## خروجی
معادل رشته‌ای عدد

## مثال
```
ورودی:
12345

خروجی:
12345
```

## راهنمایی
- از الگوریتم مناسب استفاده کنید
- حالات خاص را در نظر بگیرید
- کد خود را تست کنید
