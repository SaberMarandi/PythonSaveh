# سوال A0105 - بررسی عدد آرمسترانگ

## صورت مسئله
برنامه‌ای بنویسید که یک عدد دریافت کند و بررسی کند آیا عدد آرمسترانگ است یا خیر. (عددی که مجموع توان n ام ارقامش با خودش برابر است، n تعداد ارقام)

## ورودی
یک عدد صحیح

## خروجی
YES اگر آرمسترانگ باشد، در غیر این صورت NO

## مثال
```
ورودی:
153

خروجی:
YES
```
توضیح: 153 = 1³ + 5³ + 3³ = 1 + 125 + 27

## راهنمایی
- تعداد ارقام را بشمارید
- هر رقم را به توان تعداد ارقام برسانید
- مجموع را با عدد اصلی مقایسه کنید
