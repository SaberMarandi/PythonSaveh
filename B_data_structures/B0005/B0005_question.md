# سوال B0005 - مرتب‌سازی لیست

## صورت مسئله
برنامه‌ای بنویسید که یک لیست از اعداد را از کاربر بگیرد و آن را به ترتیب صعودی مرتب کند.

## ورودی
تعداد عناصر n، سپس n عدد صحیح

## خروجی
لیست مرتب شده

## مثال
```
ورودی:
5
3 7 2 9 4

خروجی:
[2, 3, 4, 7, 9]
```

## راهنمایی
- از تابع `sorted()` یا متد `sort()` استفاده کنید
