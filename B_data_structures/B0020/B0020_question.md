# سوال B0020 - تبدیل لیست به رشته

## صورت مسئله
برنامه‌ای بنویسید که یک لیست از اعداد را از کاربر بگیرد و آن را به یک رشته با فاصله تبدیل کند.

## ورودی
تعداد عناصر n، سپس n عدد صحیح

## خروجی
رشته حاوی اعداد با فاصله

## مثال
```
ورودی:
5
1 2 3 4 5

خروجی:
1 2 3 4 5
```

## راهنمایی
- از متد `join()` استفاده کنید
