# سوال B0023 - مرتب‌سازی لیست

## صورت مسئله
برنامه‌ای بنویسید که یک لیست از اعداد را به صورت صعودی مرتب کند.

## ورودی
یک خط شامل اعداد صحیح جدا شده با فاصله

## خروجی
لیست مرتب شده به صورت صعودی

## مثال
```
ورودی:
5 2 8 1 9 3

خروجی:
1 2 3 5 8 9
```

## راهنمایی
- از متد sort() برای مرتب‌سازی درجا استفاده کنید
- یا از تابع sorted() که لیست جدید برمی‌گرداند
- برای نزولی، از reverse=True استفاده کنید
