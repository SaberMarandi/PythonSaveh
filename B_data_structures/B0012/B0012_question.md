# سوال B0012 - برش لیست

## صورت مسئله
برنامه‌ای بنویسید که یک لیست و دو شاخص را از کاربر بگیرد و بخشی از لیست را برش دهد.

## ورودی
تعداد عناصر n، n عدد صحیح، شاخص شروع، شاخص پایان

## خروجی
بخش برش داده شده

## مثال
```
ورودی:
7
1 2 3 4 5 6 7
2
5

خروجی:
[3, 4, 5]
```

## راهنمایی
- از slicing استفاده کنید: list[start:end]
