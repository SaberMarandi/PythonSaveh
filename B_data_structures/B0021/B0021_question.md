# سوال B0021 - ایجاد تاپل

## صورت مسئله
برنامه‌ای بنویسید که تعدادی عدد را از کاربر بگیرد و آن‌ها را در یک تاپل ذخیره کند.

## ورودی
تعداد عناصر n، سپس n عدد صحیح

## خروجی
تاپل اعداد

## مثال
```
ورودی:
5
1 2 3 4 5

خروجی:
(1, 2, 3, 4, 5)
```

## راهنمایی
- تاپل غیرقابل تغییر است
- از `tuple()` برای ایجاد استفاده کنید
