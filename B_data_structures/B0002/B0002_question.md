# سوال B0002 - جمع عناصر لیست

## صورت مسئله
برنامه‌ای بنویسید که یک لیست از اعداد را از کاربر بگیرد و مجموع عناصر آن را محاسبه کند.

## ورودی
تعداد عناصر n، سپس n عدد صحیح

## خروجی
مجموع عناصر لیست

## مثال
```
ورودی:
5
1 2 3 4 5

خروجی:
15
```

## راهنمایی
- از تابع `sum()` استفاده کنید
- یا با حلقه روی عناصر جمع کنید
