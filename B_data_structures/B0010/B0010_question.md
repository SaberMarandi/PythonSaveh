# سوال B0010 - درج عنصر در لیست

## صورت مسئله
برنامه‌ای بنویسید که یک لیست، یک موقعیت و یک عدد را از کاربر بگیرد و عدد را در موقعیت مشخص شده درج کند.

## ورودی
تعداد عناصر n، n عدد صحیح، موقعیت، عدد جدید

## خروجی
لیست پس از درج

## مثال
```
ورودی:
5
1 2 3 4 5
2
10

خروجی:
[1, 2, 10, 3, 4, 5]
```

## راهنمایی
- از متد `insert()` استفاده کنید
