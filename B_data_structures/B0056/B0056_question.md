# سوال B0056 - ایجاد ست

## صورت مسئله
برنامه‌ای بنویسید که تعدادی عدد را از کاربر بگیرد و آن‌ها را در یک ست ذخیره کند (اعداد تکراری حذف شوند).

## ورودی
- خط اول: تعداد اعداد n
- خط دوم: n عدد صحیح (با فاصله)

## خروجی
ست اعداد (بدون تکرار)

## مثال
```
ورودی:
7
3 7 2 7 4 3 9

خروجی:
{2, 3, 4, 7, 9}
```

## راهنمایی
- ست با {} یا set() ایجاد می‌شود
- ست به طور خودکار اعداد تکراری را حذف می‌کند
- ترتیب عناصر در ست مهم نیست
