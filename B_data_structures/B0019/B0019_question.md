# سوال B0019 - چرخش لیست

## صورت مسئله
برنامه‌ای بنویسید که یک لیست و عدد k را از کاربر بگیرد و لیست را به اندازه k به چپ بچرخاند.

## ورودی
تعداد عناصر n، n عدد صحیح، سپس k

## خروجی
لیست چرخش یافته

## مثال
```
ورودی:
5
1 2 3 4 5
2

خروجی:
[3, 4, 5, 1, 2]
```

## راهنمایی
- از slicing استفاده کنید
