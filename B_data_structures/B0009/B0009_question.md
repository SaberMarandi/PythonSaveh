# سوال B0009 - حذف عنصر از لیست

## صورت مسئله
برنامه‌ای بنویسید که یک لیست و یک عدد را از کاربر بگیرد و اولین رخداد آن عدد را از لیست حذف کند.

## ورودی
تعداد عناصر n، n عدد صحیح، سپس عدد مورد نظر برای حذف

## خروجی
لیست پس از حذف

## مثال
```
ورودی:
5
1 2 3 4 5
3

خروجی:
[1, 2, 4, 5]
```

## راهنمایی
- از متد `remove()` استفاده کنید
