# سوال B0025 - شمارش تکرار

## صورت مسئله
برنامه‌ای بنویسید که تعداد تکرار هر عنصر در لیست را بشمارد و نمایش دهد.

## ورودی
یک خط شامل اعداد صحیح جدا شده با فاصله

## خروجی
هر عدد و تعداد تکرار آن در خط جداگانه

## مثال
```
ورودی:
1 2 3 2 1 2 4

خروجی:
1: 2
2: 3
3: 1
4: 1
```

## راهنمایی
- از دیکشنری برای نگهداری تعداد استفاده کنید
- یا از ماژول collections.Counter
- متد count() هم می‌تواند کمک کند
