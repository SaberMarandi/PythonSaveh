# سوال B0024 - پیدا کردن عنصر

## صورت مسئله
برنامه‌ای بنویسید که بررسی کند عنصر x در لیست وجود دارد یا خیر.

## ورودی
- خط اول: لیست اعداد جدا شده با فاصله
- خط دوم: عدد x برای جستجو

## خروجی
"Yes" اگر x در لیست باشد، در غیر این صورت "No"

## مثال
```
ورودی:
1 5 3 8 2
5

خروجی:
Yes
```

```
ورودی:
1 5 3 8 2
7

خروجی:
No
```

## راهنمایی
- از عملگر in برای بررسی عضویت استفاده کنید
- یا با حلقه جستجو کنید
