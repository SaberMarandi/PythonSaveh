# سوال B0008 - شمارش تکرار عنصر

## صورت مسئله
برنامه‌ای بنویسید که یک لیست و یک عدد را از کاربر بگیرد و تعداد تکرار آن عدد در لیست را بشمارد.

## ورودی
تعداد عناصر n، n عدد صحیح، سپس عدد مورد نظر

## خروجی
تعداد تکرار عدد

## مثال
```
ورودی:
7
1 2 3 2 4 2 5
2

خروجی:
3
```

## راهنمایی
- از متد `count()` استفاده کنید
