# سوال B0031 - ایجاد دیکشنری

## صورت مسئله
برنامه‌ای بنویسید که 3 جفت کلید-مقدار را از کاربر بگیرد و یک دیکشنری ایجاد کند.

## ورودی
- 3 خط، هر خط شامل یک کلید و یک مقدار (با فاصله)

## خروجی
دیکشنری ایجاد شده

## مثال
```
ورودی:
name Ali
age 25
city Tehran

خروجی:
{'name': 'Ali', 'age': '25', 'city': 'Tehran'}
```

## راهنمایی
- دیکشنری با {} ایجاد می‌شود
- از متد split() برای جدا کردن کلید و مقدار استفاده کنید
