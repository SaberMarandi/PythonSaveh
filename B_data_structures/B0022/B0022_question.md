# سوال B0022 - حذف تکراری از لیست

## صورت مسئله
برنامه‌ای بنویسید که عناصر تکراری یک لیست را حذف کند و فقط عناصر یکتا را نگه دارد.

## ورودی
یک خط شامل اعداد صحیح جدا شده با فاصله

## خروجی
لیست بدون تکرار (ترتیب مهم نیست)

## مثال
```
ورودی:
1 2 3 2 4 1 5

خروجی:
1 2 3 4 5
```

## راهنمایی
- می‌توانید از set برای حذف تکراری استفاده کنید
- یا با حلقه و بررسی عضویت
- توجه کنید که set ترتیب را حفظ نمی‌کند
