# سوال B0022 - ایجاد تاپل

## صورت مسئله
برنامه‌ای بنویسید که 3 عدد را از کاربر بگیرد و آن‌ها را در یک تاپل ذخیره کرده و چاپ کند.

## ورودی
3 عدد صحیح (هر کدام در یک خط)

## خروجی
تاپل اعداد

## مثال
```
ورودی:
10
20
30

خروجی:
(10, 20, 30)
```

## راهنمایی
- تاپل با پرانتز () ایجاد می‌شود
- تاپل غیرقابل تغییر است
- می‌توانید از tuple() برای تبدیل لیست به تاپل استفاده کنید
