# تمرین‌های فصل 2 🎮

## تمرین 1: سلام به خودت! 👋

یه برنامه بنویس که اسمت رو 5 بار چاپ کنه.

**مثال خروجی:**
```
علی
علی
علی
علی
علی
```

**راهنمایی:** از `print()` استفاده کن!

---

## تمرین 2: ماشین حساب ساده 🔢

یه برنامه بنویس که دو عدد رو جمع کنه.

**مثال:**
```
عدد اول: 5
عدد دوم: 3
جواب: 8
```

**راهنمایی:**
```python
num1 = int(input("عدد اول: "))
num2 = int(input("عدد دوم: "))
# حالا جمعشون کن!
```

---

## تمرین 3: معرفی خودت 🌟

یه برنامه بنویس که این اطلاعات رو از کاربر بگیره و نمایش بده:
- اسم
- سن
- شهر
- مدرسه

**مثال خروجی:**
```
اسم: علی
سن: 14 سال
شهر: تهران
مدرسه: دبیرستان شهید بهشتی
```

---

## تمرین 4: جدول ضرب 📊

یه برنامه بنویس که جدول ضرب 2 رو نمایش بده.

**خروجی:**
```
2 × 1 = 2
2 × 2 = 4
2 × 3 = 6
2 × 4 = 8
2 × 5 = 10
```

**راهنمایی:** از `print()` برای هر خط استفاده کن.

---

## تمرین 5: کارت تبریک 🎂

یه برنامه بنویس که یه کارت تبریک تولد قشنگ بسازه!

**مثال خروجی:**
```
***********************
*                     *
*   تولدت مبارک! 🎉   *
*                     *
***********************
```

**راهنمایی:** از `*` و فاصله استفاده کن.

---

## تمرین 6: تبدیل سن به روز 📅

یه برنامه بنویس که سن کاربر رو بگیره و بگه چند روز زندگی کرده!

**مثال:**
```
چند سالته؟ 14
تو تقریباً 5110 روز زندگی کردی! 🎊
```

**راهنمایی:** یه سال حدوداً 365 روزه.

---

## تمرین 7: محاسبه مساحت 📐

یه برنامه بنویس که طول و عرض یه مستطیل رو بگیره و مساحتش رو حساب کنه.

**مثال:**
```
طول: 5
عرض: 3
مساحت: 15
```

**راهنمایی:** مساحت = طول × عرض

---

## تمرین 8: تکرار کلمه 🔁

یه برنامه بنویس که یه کلمه و یه عدد بگیره، و اون کلمه رو به تعداد اون عدد تکرار کنه.

**مثال:**
```
کلمه: پایتون
تعداد: 3
خروجی: پایتون پایتون پایتون
```

**راهنمایی:** می‌تونی از `*` برای تکرار استفاده کنی!

---

## تمرین 9: خط فاصل 📏

یه برنامه بنویس که یه خط فاصل قشنگ بسازه.

**خروجی:**
```
================================
        پایتون جالبه!
================================
```

**راهنمایی:** از `=` و `*` استفاده کن.

---

## تمرین 10: برنامه دوستانه 🤝

یه برنامه بنویس که:
1. اسم کاربر رو بگیره
2. بگه "سلام [اسم]!"
3. بپرسه "حالت چطوره؟"
4. جواب رو بگیره
5. بگه "خوشحالم که [حالت] هستی!"

**مثال:**
```
اسمت چیه؟ علی
سلام علی!
حالت چطوره؟ خوب
خوشحالم که خوب هستی!
```

---

## 🌟 پروژه چالشی: کارت ویزیت

یه برنامه بنویس که یه کارت ویزیت قشنگ برای خودت بسازه!

باید شامل:
- اسم
- سن
- شهر
- ایمیل (اختیاری)
- علاقه‌مندی‌ها

**مثال:**
```
╔════════════════════════════╗
║     کارت ویزیت من         ║
╠════════════════════════════╣
║ اسم: علی احمدی             ║
║ سن: 14 سال                ║
║ شهر: تهران                ║
║ علاقه: برنامه‌نویسی 💻     ║
╚════════════════════════════╝
```

**نکته:** می‌تونی از کاراکترهای خاص استفاده کنی!
