# سوال D0035 - جایگزینی کلمه در فایل

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلمه را در فایل با کلمه دیگر جایگزین کند.

## ورودی
- خط اول: نام فایل ورودی
- خط دوم: کلمه قدیمی
- خط سوم: کلمه جدید
- خط چهارم: نام فایل خروجی

## خروجی
پیام موفقیت

## مثال
```
ورودی:
text.txt
old
new
updated.txt

خروجی:
Replaced successfully
```

## راهنمایی
- فایل را بخوانید
- از replace() استفاده کنید
- در فایل جدید بنویسید
