# سوال D0037 - تقسیم فایل

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک فایل را به چند فایل کوچکتر تقسیم کند (هر فایل n خط).

## ورودی
- خط اول: نام فایل ورودی
- خط دوم: تعداد خطوط در هر فایل n

## خروجی
پیام موفقیت و تعداد فایل‌های ایجاد شده

## مثال
```
ورودی:
large.txt
100

خروجی:
Split into 5 files
```

## راهنمایی
- خطوط را بخوانید
- هر n خط را در فایل جداگانه بنویسید
- نام فایل‌ها: part1.txt, part2.txt, ...
