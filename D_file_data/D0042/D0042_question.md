# سوال D0042 - حذف پوشه خالی

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک پوشه خالی را حذف کند.

## ورودی
نام پوشه

## خروجی
پیام موفقیت یا خطا

## مثال
```
ورودی:
temp_folder

خروجی:
Folder deleted successfully
```

## راهنمایی
- از os.rmdir() استفاده کنید
- فقط پوشه‌های خالی را حذف می‌کند
