# سوال D0048 - کپی پوشه

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک پوشه و تمام محتویات آن را کپی کند.

## ورودی
- خط اول: مسیر پوشه مبدا
- خط دوم: مسیر پوشه مقصد

## خروجی
پیام موفقیت

## مثال
```
ورودی:
source_folder
destination_folder

خروجی:
Folder copied successfully
```

## راهنمایی
- از shutil.copytree() استفاده کنید
