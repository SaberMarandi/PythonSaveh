# سوال D0012 - شمارش کلمات فایل

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که تعداد کلمات یک فایل را بشمارد.

## ورودی
نام فایل

## خروجی
تعداد کلمات

## مثال
```
ورودی:
document.txt

خروجی:
1250
```

## راهنمایی
- فایل را بخوانید
- با split() کلمات را جدا کنید
- از len() برای شمارش استفاده کنید
