# سوال D0030 - مرتب‌سازی خطوط فایل

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که خطوط یک فایل را به ترتیب الفبایی مرتب کند.

## ورودی
- خط اول: نام فایل ورودی
- خط دوم: نام فایل خروجی

## خروجی
پیام موفقیت

## مثال
```
ورودی:
names.txt
sorted_names.txt

خروجی:
Lines sorted successfully
```

## راهنمایی
- خطوط را بخوانید
- از sorted() استفاده کنید
- در فایل جدید بنویسید
