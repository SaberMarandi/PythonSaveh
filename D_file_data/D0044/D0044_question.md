# سوال D0044 - شمارش فایل‌های پوشه

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که تعداد فایل‌ها و پوشه‌های یک مسیر را بشمارد.

## ورودی
مسیر پوشه

## خروجی
- خط اول: تعداد فایل‌ها
- خط دوم: تعداد پوشه‌ها

## مثال
```
ورودی:
documents

خروجی:
15 files
3 folders
```

## راهنمایی
- از os.listdir() استفاده کنید
- فایل‌ها و پوشه‌ها را جدا بشمارید
