# سوال D0005 - افزودن به فایل

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که متن جدید را به انتهای فایل اضافه کند.

## ورودی
نام فایل و متن جدید

## خروجی
پیام موفقیت

## مثال
```
ورودی:
output.txt
New line

خروجی:
Text appended successfully
```

## راهنمایی
- از mode='a' برای append استفاده کنید
