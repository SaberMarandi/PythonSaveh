# سوال D0049 - حذف پوشه و محتویات

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک پوشه و تمام محتویات آن را حذف کند.

## ورودی
مسیر پوشه

## خروجی
پیام موفقیت

## مثال
```
ورودی:
temp_folder

خروجی:
Folder deleted successfully
```

## راهنمایی
- از shutil.rmtree() استفاده کنید
- احتیاط کنید! این عملیات برگشت‌ناپذیر است
