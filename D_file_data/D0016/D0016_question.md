# سوال D0016 - شمارش خطوط خالی

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که تعداد خطوط خالی یک فایل را بشمارد.

## ورودی
نام فایل

## خروجی
تعداد خطوط خالی

## مثال
```
ورودی:
document.txt

خروجی:
15
```

## راهنمایی
- خطوط را بخوانید
- بررسی کنید که آیا خط خالی است (strip() == '')
- تعداد را بشمارید
