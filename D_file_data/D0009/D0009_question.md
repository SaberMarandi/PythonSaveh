# سوال D0009 - تغییر نام فایل

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که نام یک فایل را تغییر دهد.

## ورودی
- خط اول: نام فعلی فایل
- خط دوم: نام جدید فایل

## خروجی
پیام موفقیت یا خطا

## مثال
```
ورودی:
old_name.txt
new_name.txt

خروجی:
File renamed successfully
```

## راهنمایی
- از os.rename() استفاده کنید
- خطاها را مدیریت کنید
