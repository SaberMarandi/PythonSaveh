# سوال D0024 - شمارش تکرار کلمات

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که فراوانی هر کلمه در فایل را بشمارد و نمایش دهد.

## ورودی
نام فایل

## خروجی
هر کلمه و تعداد تکرار آن (مرتب شده)

## مثال
```
ورودی:
text.txt

خروجی:
python: 5
programming: 3
language: 2
```

## راهنمایی
- فایل را بخوانید
- کلمات را جدا کنید
- از دیکشنری برای شمارش استفاده کنید
