# سوال D0007 - خواندن فایل

## صورت مسئله
برنامه‌ای بنویسید که محتوای کامل یک فایل متنی را بخواند و روی صفحه چاپ کند.

## ورودی
نام فایل

## خروجی
محتوای کامل فایل

## مثال
```
ورودی:
sample.txt

خروجی:
[محتوای فایل sample.txt]
```

## راهنمایی
- از with open برای باز کردن فایل استفاده کنید
- متد read() کل محتوا را برمی‌گرداند
- readlines() لیستی از خطوط برمی‌گرداند
- encoding='utf-8' را فراموش نکنید
