# سوال D0007 - کپی فایل

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک فایل را کپی کند.

## ورودی
- خط اول: نام فایل مبدا
- خط دوم: نام فایل مقصد

## خروجی
پیام موفقیت یا خطا

## مثال
```
ورودی:
source.txt
destination.txt

خروجی:
File copied successfully
```

## راهنمایی
- از ماژول shutil استفاده کنید
- یا محتوا را بخوانید و در فایل جدید بنویسید
