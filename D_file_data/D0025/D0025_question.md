# سوال D0025 - فیلتر خطوط حاوی عدد

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که فقط خطوطی که حاوی عدد هستند را در فایل جدید ذخیره کند.

## ورودی
- خط اول: نام فایل ورودی
- خط دوم: نام فایل خروجی

## خروجی
پیام موفقیت

## مثال
```
ورودی:
data.txt
numbers.txt

خروجی:
Filtered successfully
```

## راهنمایی
- خطوط را بخوانید
- بررسی کنید که آیا خط حاوی عدد است
- از isdigit() استفاده کنید
