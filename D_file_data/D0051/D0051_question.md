# سوال D0051 - جستجو در فایل

## سطح: متوسط

## صورت مسئله
برنامه‌ای بنویسید که یک کلمه را در فایل جستجو کند و تعداد تکرار آن را بشمارد.

## ورودی
نام فایل و کلمه مورد جستجو

## خروجی
تعداد تکرار کلمه

## مثال
```
ورودی:
input.txt
python

خروجی:
5
```

## راهنمایی
- فایل را بخوانید و به کلمات تقسیم کنید
- از متد `count()` استفاده کنید
