# سوال D0023 - تبدیل حروف به بزرگ

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که تمام حروف یک فایل را به حروف بزرگ تبدیل کند.

## ورودی
- خط اول: نام فایل ورودی
- خط دوم: نام فایل خروجی

## خروجی
پیام موفقیت

## مثال
```
ورودی:
input.txt
output.txt

خروجی:
Converted successfully
```

## راهنمایی
- فایل را بخوانید
- از upper() استفاده کنید
- در فایل جدید بنویسید
