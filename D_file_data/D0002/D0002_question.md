# سوال D0002 - نوشتن در فایل

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک متن را از کاربر بگیرد و در فایل بنویسد.

## ورودی
نام فایل و متن

## خروجی
پیام موفقیت

## مثال
```
ورودی:
output.txt
Hello World

خروجی:
File written successfully
```

## راهنمایی
- از mode='w' برای نوشتن استفاده کنید
