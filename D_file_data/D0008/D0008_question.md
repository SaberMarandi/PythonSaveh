# سوال D0008 - حذف فایل

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک فایل را حذف کند.

## ورودی
نام فایل

## خروجی
پیام موفقیت یا خطا

## مثال
```
ورودی:
temp.txt

خروجی:
File deleted successfully
```

## راهنمایی
- از ماژول os استفاده کنید
- از os.remove() برای حذف فایل استفاده کنید
- خطاها را مدیریت کنید
