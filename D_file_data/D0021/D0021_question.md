# سوال D0021 - فیلتر خطوط بلندتر از n

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که خطوطی که طول آنها بیشتر از n کاراکتر است را در فایل جدید ذخیره کند.

## ورودی
- خط اول: نام فایل ورودی
- خط دوم: عدد n
- خط سوم: نام فایل خروجی

## خروجی
پیام موفقیت

## مثال
```
ورودی:
input.txt
50
output.txt

خروجی:
Filtered successfully
```

## راهنمایی
- خطوط را بخوانید
- طول هر خط را بررسی کنید
- خطوط بلند را در فایل جدید بنویسید
