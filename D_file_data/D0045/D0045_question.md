# سوال D0045 - لیست فایل‌های پوشه

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که تمام فایل‌های یک پوشه را لیست کند.

## ورودی
نام پوشه

## خروجی
لیست فایل‌ها (هر فایل در یک خط)

## مثال
```
ورودی:
documents

خروجی:
file1.txt
file2.pdf
image.jpg
```

## راهنمایی
- از os.listdir() استفاده کنید
- یا از pathlib.Path استفاده کنید
- فقط فایل‌ها را نمایش دهید (نه پوشه‌ها)
