# سوال D0046 - جستجوی فایل در پوشه

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که بررسی کند آیا فایلی با نام خاص در پوشه وجود دارد یا خیر.

## ورودی
- خط اول: مسیر پوشه
- خط دوم: نام فایل

## خروجی
YES یا NO

## مثال
```
ورودی:
documents
report.pdf

خروجی:
YES
```

## راهنمایی
- از os.listdir() استفاده کنید
- یا از os.path.exists() استفاده کنید
