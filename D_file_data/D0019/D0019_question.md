# سوال D0019 - معکوس کردن خطوط فایل

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که خطوط یک فایل را به ترتیب معکوس در فایل جدید بنویسد.

## ورودی
- خط اول: نام فایل ورودی
- خط دوم: نام فایل خروجی

## خروجی
پیام موفقیت

## مثال
```
ورودی:
input.txt
output.txt

خروجی:
Lines reversed successfully
```

## راهنمایی
- خطوط را بخوانید
- لیست را معکوس کنید
- در فایل جدید بنویسید
