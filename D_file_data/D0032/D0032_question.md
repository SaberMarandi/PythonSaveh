# سوال D0032 - استخراج ایمیل‌ها

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که تمام آدرس‌های ایمیل یک فایل را استخراج کند.

## ورودی
نام فایل

## خروجی
لیست ایمیل‌ها (هر ایمیل در یک خط)

## مثال
```
ورودی:
contacts.txt

خروجی:
user@example.com
admin@site.org
```

## راهنمایی
- از ماژول re استفاده کنید
- الگوی regex برای ایمیل: \S+@\S+
- از re.findall() استفاده کنید
