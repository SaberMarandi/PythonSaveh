# سوال D0006 - شمارش خطوط فایل

## صورت مسئله
برنامه‌ای بنویسید که تعداد خطوط یک فایل متنی را بشمارد.

## ورودی
نام فایل (رشته)

## خروجی
تعداد خطوط فایل

## مثال
```
ورودی:
test.txt

خروجی:
15
```

## راهنمایی
- از دستور with open برای باز کردن فایل استفاده کنید
- متد readlines() لیستی از خطوط برمی‌گرداند
- یا می‌توانید خطوط را در حلقه بشمارید
- فراموش نکنید encoding='utf-8' را مشخص کنید
