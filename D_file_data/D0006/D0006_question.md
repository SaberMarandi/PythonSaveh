# سوال D0006 - خواندن تمام خطوط فایل

## صورت مسئله
برنامه‌ای بنویسید که تمام خطوط یک فایل را بخواند و تعداد خطوط را چاپ کند.

## ورودی
نام فایل

## خروجی
تعداد خطوط فایل

## مثال
```
ورودی:
data.txt

خروجی:
10
```

## راهنمایی
- از متد readlines() استفاده کنید
- یا از حلقه for برای پیمایش خطوط استفاده کنید
- از len() برای شمارش استفاده کنید
