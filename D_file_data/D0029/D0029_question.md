# سوال D0029 - شمارش خطوط حاوی کلمه

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که تعداد خطوطی که حاوی یک کلمه خاص هستند را بشمارد.

## ورودی
- خط اول: نام فایل
- خط دوم: کلمه جستجو

## خروجی
تعداد خطوط

## مثال
```
ورودی:
log.txt
error

خروجی:
12
```

## راهنمایی
- خطوط را بخوانید
- بررسی کنید که آیا کلمه در خط وجود دارد
- تعداد را بشمارید
