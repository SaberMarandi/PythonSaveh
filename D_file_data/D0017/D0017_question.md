# سوال D0017 - حذف خطوط خالی

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که خطوط خالی یک فایل را حذف کند و نتیجه را در فایل جدید ذخیره کند.

## ورودی
- خط اول: نام فایل ورودی
- خط دوم: نام فایل خروجی

## خروجی
پیام موفقیت

## مثال
```
ورودی:
input.txt
output.txt

خروجی:
Empty lines removed successfully
```

## راهنمایی
- خطوط را بخوانید
- فقط خطوط غیرخالی را نگه دارید
- در فایل جدید بنویسید
