# سوال D0034 - حذف کامنت‌ها

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که خطوط کامنت (شروع با #) را از فایل حذف کند.

## ورودی
- خط اول: نام فایل ورودی
- خط دوم: نام فایل خروجی

## خروجی
پیام موفقیت

## مثال
```
ورودی:
code.py
clean.py

خروجی:
Comments removed successfully
```

## راهنمایی
- خطوط را بخوانید
- خطوطی که با # شروع نمی‌شوند را نگه دارید
- در فایل جدید بنویسید
