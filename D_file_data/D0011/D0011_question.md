# سوال D0011 - جستجوی کلمه در فایل

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک کلمه را در فایل جستجو کند و تعداد تکرار آن را برگرداند.

## ورودی
- خط اول: نام فایل
- خط دوم: کلمه جستجو

## خروجی
تعداد تکرار کلمه

## مثال
```
ورودی:
data.txt
python

خروجی:
5
```

## راهنمایی
- فایل را بخوانید
- از متد count() استفاده کنید
- یا با split() کلمات را جدا کنید
