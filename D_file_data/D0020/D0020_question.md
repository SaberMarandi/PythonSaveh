# سوال D0020 - ادغام دو فایل

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که محتوای دو فایل را ادغام کند و در فایل سوم ذخیره کند.

## ورودی
- خط اول: نام فایل اول
- خط دوم: نام فایل دوم
- خط سوم: نام فایل خروجی

## خروجی
پیام موفقیت

## مثال
```
ورودی:
file1.txt
file2.txt
merged.txt

خروجی:
Files merged successfully
```

## راهنمایی
- هر دو فایل را بخوانید
- محتوا را ترکیب کنید
- در فایل جدید بنویسید
