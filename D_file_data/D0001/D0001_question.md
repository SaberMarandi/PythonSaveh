# سوال D0001 - خواندن فایل متنی

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که محتوای یک فایل متنی را بخواند و چاپ کند.

## ورودی
نام فایل

## خروجی
محتوای فایل

## مثال
```
ورودی:
input.txt

خروجی:
(محتوای فایل)
```

## راهنمایی
- از `open()` و `read()` استفاده کنید
- از `with` برای مدیریت فایل استفاده کنید
