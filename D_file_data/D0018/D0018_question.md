# سوال D0018 - شماره‌گذاری خطوط

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که خطوط یک فایل را با شماره بخواند و چاپ کند.

## ورودی
نام فایل

## خروجی
خطوط با شماره

## مثال
```
ورودی:
code.py

خروجی:
1: import os
2: import sys
3: 
4: def main():
```

## راهنمایی
- از enumerate() استفاده کنید
- شماره خط را قبل از محتوا چاپ کنید
