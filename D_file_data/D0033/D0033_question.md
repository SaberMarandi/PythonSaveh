# سوال D0033 - حذف فاصله‌های اضافی

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که فاصله‌های اضافی (بیش از یک فاصه) را از فایل حذف کند.

## ورودی
- خط اول: نام فایل ورودی
- خط دوم: نام فایل خروجی

## خروجی
پیام موفقیت

## مثال
```
ورودی:
input.txt
output.txt

خروجی:
Spaces removed successfully
```

## راهنمایی
- از regex استفاده کنید
- الگوی \s+ را با یک فاصه جایگزین کنید
