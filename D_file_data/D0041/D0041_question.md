# سوال D0041 - ایجاد پوشه

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که یک پوشه جدید ایجاد کند.

## ورودی
نام پوشه

## خروجی
پیام موفقیت یا خطا

## مثال
```
ورودی:
new_folder

خروجی:
Folder created successfully
```

## راهنمایی
- از os.mkdir() استفاده کنید
- یا از pathlib.Path().mkdir() استفاده کنید
- خطاها را مدیریت کنید
