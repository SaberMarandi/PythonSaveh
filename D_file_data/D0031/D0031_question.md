# سوال D0031 - جایگزینی با regex

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که با استفاده از regex، تمام اعداد یک فایل را با کلمه "NUMBER" جایگزین کند.

## ورودی
- خط اول: نام فایل ورودی
- خط دوم: نام فایل خروجی

## خروجی
پیام موفقیت

## مثال
```
ورودی:
data.txt
output.txt

خروجی:
Replaced successfully
```

## راهنمایی
- از ماژول re استفاده کنید
- از re.sub() برای جایگزینی استفاده کنید
- الگوی \d+ برای اعداد
