# سوال D0043 - لیست پوشه‌ها

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که تمام پوشه‌های یک مسیر را لیست کند.

## ورودی
مسیر پوشه

## خروجی
لیست پوشه‌ها (هر پوشه در یک خط)

## مثال
```
ورودی:
.

خروجی:
folder1
folder2
subfolder
```

## راهنمایی
- از os.listdir() استفاده کنید
- فقط پوشه‌ها را نمایش دهید
- از os.path.isdir() برای بررسی استفاده کنید
