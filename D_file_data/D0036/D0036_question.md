# سوال D0036 - ترکیب چند فایل

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که محتوای چند فایل را در یک فایل ترکیب کند.

## ورودی
- خط اول: تعداد فایل‌ها n
- n خط بعدی: نام فایل‌ها
- خط آخر: نام فایل خروجی

## خروجی
پیام موفقیت

## مثال
```
ورودی:
3
file1.txt
file2.txt
file3.txt
combined.txt

خروجی:
Files combined successfully
```

## راهنمایی
- هر فایل را بخوانید
- محتوا را به فایل خروجی اضافه کنید
