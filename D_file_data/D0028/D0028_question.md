# سوال D0028 - حذف خطوط تکراری

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که خطوط تکراری یک فایل را حذف کند و فقط یک نسخه از هر خط را نگه دارد.

## ورودی
- خط اول: نام فایل ورودی
- خط دوم: نام فایل خروجی

## خروجی
پیام موفقیت

## مثال
```
ورودی:
input.txt
unique.txt

خروجی:
Duplicates removed successfully
```

## راهنمایی
- خطوط را بخوانید
- از set یا dict.fromkeys() برای حذف تکراری استفاده کنید
- ترتیب را حفظ کنید
