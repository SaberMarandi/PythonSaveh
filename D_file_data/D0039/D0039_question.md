# سوال D0039 - تاریخ آخرین تغییر

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که تاریخ و زمان آخرین تغییر یک فایل را نمایش دهد.

## ورودی
نام فایل

## خروجی
تاریخ و زمان آخرین تغییر

## مثال
```
ورودی:
document.txt

خروجی:
2024-01-15 14:30:25
```

## راهنمایی
- از os.path.getmtime() استفاده کنید
- از datetime برای فرمت کردن استفاده کنید
