# سوال D0040 - مقایسه دو فایل

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که بررسی کند آیا دو فایل یکسان هستند یا خیر.

## ورودی
- خط اول: نام فایل اول
- خط دوم: نام فایل دوم

## خروجی
"Files are identical" یا "Files are different"

## مثال
```
ورودی:
file1.txt
file2.txt

خروجی:
Files are identical
```

## راهنمایی
- هر دو فایل را بخوانید
- محتوا را مقایسه کنید
- یا از filecmp.cmp() استفاده کنید
