# سوال D0014 - خواندن خط خاص

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که خط n ام یک فایل را بخواند و چاپ کند.

## ورودی
- خط اول: نام فایل
- خط دوم: شماره خط (از 1 شروع)

## خروجی
محتوای خط n ام

## مثال
```
ورودی:
data.txt
5

خروجی:
(محتوای خط پنجم)
```

## راهنمایی
- تمام خطوط را بخوانید
- خط مورد نظر را با اندیس انتخاب کنید
