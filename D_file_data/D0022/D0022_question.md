# سوال D0022 - فیلتر خطوط شروع با کاراکتر

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که فقط خطوطی که با یک کاراکتر خاص شروع می‌شوند را در فایل جدید ذخیره کند.

## ورودی
- خط اول: نام فایل ورودی
- خط دوم: کاراکتر شروع
- خط سوم: نام فایل خروجی

## خروجی
پیام موفقیت

## مثال
```
ورودی:
code.py
#
comments.txt

خروجی:
Filtered successfully
```

## راهنمایی
- خطوط را بخوانید
- از startswith() استفاده کنید
- خطوط مطابق را در فایل جدید بنویسید
