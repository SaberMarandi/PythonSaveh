# سوال D0026 - یافتن طولانی‌ترین خط

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که طولانی‌ترین خط یک فایل را پیدا کند و چاپ کند.

## ورودی
نام فایل

## خروجی
طولانی‌ترین خط

## مثال
```
ورودی:
data.txt

خروجی:
This is the longest line in the file with many characters
```

## راهنمایی
- خطوط را بخوانید
- طول هر خط را بررسی کنید
- بزرگترین را نگه دارید
