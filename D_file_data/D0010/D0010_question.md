# سوال D0010 - بررسی وجود فایل

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که بررسی کند آیا یک فایل وجود دارد یا خیر.

## ورودی
نام فایل

## خروجی
YES اگر فایل وجود داشته باشد، در غیر این صورت NO

## مثال
```
ورودی:
data.txt

خروجی:
YES
```

## راهنمایی
- از os.path.exists() استفاده کنید
- یا از pathlib.Path استفاده کنید
