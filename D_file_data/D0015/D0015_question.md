# سوال D0015 - جستجوی خط حاوی کلمه

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که تمام خطوطی که حاوی یک کلمه خاص هستند را چاپ کند.

## ورودی
- خط اول: نام فایل
- خط دوم: کلمه جستجو

## خروجی
خطوط حاوی کلمه (هر خط در یک خط)

## مثال
```
ورودی:
log.txt
error

خروجی:
Line 5: Error occurred
Line 12: Critical error
```

## راهنمایی
- خطوط را یکی یکی بخوانید
- بررسی کنید که آیا کلمه در خط وجود دارد
- از عملگر in استفاده کنید
