# سوال D0050 - خواندن n خط اول فایل بزرگ

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که n خط اول یک فایل بزرگ را بخواند (بدون بارگذاری کل فایل در حافظه).

## ورودی
- خط اول: نام فایل
- خط دوم: تعداد خطوط n

## خروجی
n خط اول فایل

## مثال
```
ورودی:
large_file.txt
10

خروجی:
(10 خط اول فایل)
```

## راهنمایی
- از حلقه for برای پیمایش فایل استفاده کنید
- بعد از n خط، حلقه را break کنید
- این روش برای فایل‌های بزرگ بهینه است
