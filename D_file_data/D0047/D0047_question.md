# سوال D0047 - فایل‌های با پسوند خاص

## سطح: ساده

## صورت مسئله
برنامه‌ای بنویسید که تمام فایل‌های با پسوند خاص در یک پوشه را لیست کند.

## ورودی
- خط اول: مسیر پوشه
- خط دوم: پسوند (مثلاً .txt)

## خروجی
لیست فایل‌ها

## مثال
```
ورودی:
documents
.pdf

خروجی:
report.pdf
manual.pdf
```

## راهنمایی
- از os.listdir() استفاده کنید
- از endswith() برای بررسی پسوند استفاده کنید
